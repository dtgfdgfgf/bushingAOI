## 什麼是新料號參數設定

### 🎯 功能目的
當您的工廠要檢測一個**新的產品型號**（我們稱為「新料號」）時，需要告訴 AOI 檢測系統：
- 相機要怎麼拍照（曝光、增益）
- 產品在影像中的位置在哪裡
- 要用什麼標準來判斷產品合格或不合格

這個功能就是讓您設定這些參數的地方。

### 🔄 為什麼要「參考舊料號」
因為新產品通常和舊產品很相似，所以我們可以：
1. 選一個**相似的舊料號**當作「參考範本」
2. **複製**它的參數
3. 只**修改**不一樣的地方
4. **儲存**成新料號的參數

這樣比從零開始設定快很多！

## 開始之前的準備

### ✅ 必備資料清單

在開始設定之前，請先準備好以下資料：

#### 1. **新料號基本資訊**
- [ ] 新料號編號（例如：`10215112360T`）
- [ ] 新料號各種特徵，在新建料號時會輸入

#### 2. **參考料號選擇**
- [ ] 找一個**最相似**的舊料號（產品外型、尺寸接近、PTFE面同色）
- [ ] 確認舊料號的參數在系統中**運作正常**

#### 3. **測試樣品**
- [ ] 準備 **5-10 個良品樣品**（用於測試和校準）
- [ ] 準備 **3-5 個不良品樣品**（用於驗證缺陷檢測）

#### 4. **設備狀態**
- [ ] 確認相機已連接並正常運作
- [ ] 確認 PLC 連線正常
- [ ] 確認照明設備正常

#### 5. **AI 模型檔案（如果需要）**
- [ ] 準備好新料號的 TensorRT 模型檔案（`.pt` 檔案）
- [ ] 模型檔案命名規則：`{料號}_{in|out|1|2}.pt`
- [ ] 放置位置：`bin\x64\Release\models\` 資料夾

當您打開「參數設定」視窗時，會看到：

```

│  狀態: ✅相機(3/3) ⚠️位置(5/8) 🔧檢測(需校準) ❌時間(0/4)       │
├─────────────────────────────────────────────────────────────────┤
│ [相機參數] [位置參數] [檢測參數] [時間參數] [測試驗證]              │

│ │(來源料號)  │  │            │  │            │                 │

```

### 📊 三區域架構詳解

#### **區域 1：參考區（來源料號）**
- **位置**：最左邊
- **顏色標示**：
  - ⬜ **白色背景**：這個參數還沒被複製
  - 🟢 **淺綠色背景**：這個參數已被複製到「已新增未修改區」
  - 🔵 **淺藍色背景**：這個參數已被複製且已修改
- **功能**：顯示舊料號的所有參數，供您參考和複製
- **特性**：**唯讀**，不能修改，也不會因為您的操作而消失

#### **區域 2：已新增未修改區**
- **位置**：中間
- **功能**：放置從參考區複製過來、**但還沒修改**的參數
- **操作**：您可以直接在這裡編輯參數值

#### **區域 3：已新增已修改區**
- **位置**：最右邊
- **功能**：放置**已經修改過**的參數
- **重要**：只有這個區域的參數會被儲存到資料庫！

### 🎨 顏色與圖示說明

#### **參數狀態圖示**
- ✅ **已完成**：該分類的參數已全部設定完成
- ⚠️ **進行中**：該分類的參數正在設定中
- ❌ **未開始**：該分類的參數尚未開始設定
- 🔧 **需要外部工具**：該分類需要使用校準工具

#### **背景顏色意義**
- ⬜ **白色**：參考區中未被使用的參數
- 🟢 **淺綠色**：參考區中已複製但未修改的參數
- 🔵 **淺藍色**：參考區中已複製且已修改的參數

## 完整操作流程

### 📝 第一步：新增料號

#### 方法一：從主視窗開啟（新增料號）
1. 在主視窗中，點擊「料號設定」選單
2. 進行料號新增
3. 輸入新料號資訊
4. 點擊「儲存」後，系統有了此料號的基本資訊，可開始設定此料號的參數

---

### 📝 第二步：選擇來源料號（參考範本）

1. 在視窗上方找到「來源料號」下拉選單
   ```
   來源料號: [選擇舊料號 ▼]
   ```

2. 點擊下拉選單，會顯示所有已設定的料號清單：
   ```
   ▼ 選擇來源料號
   ├─ 10201120714TP
   ├─ 10214090420T
   ├─ 10215112360T   ← 假設選這個
   └─ ...
   ```

3. **如何選擇合適的來源料號？**
   - ✅ 選擇**產品外型最相似**的料號
   - ✅ 選擇**尺寸接近**的料號
   - ✅ 選擇**檢測需求相似**的料號
   - ✅ 選擇**PTFE同色**的料號
   - ❌ 避免選擇完全不同類型的產品

4. 選擇後，「參考區」會立即載入該料號的所有參數

---

### 📝 第三步：逐一設定五大分類參數

系統將參數分為五大類別，建議按照以下順序設定：

```
1. 相機參數 → 2. 位置參數 → 3. 檢測參數 → 4. 時間參數 → 5. 測試驗證
```

#### **為什麼要按照這個順序？**
- **相機參數**影響影像品質，必須先設定
- **位置參數**決定檢測區域，需要好的影像才能校準
- **檢測參數**各種檢測細節使用的參數
- **時間參數**影響推料時機的參數
- **測試驗證**是最後確認所有設定是否可用

#### **每個分類的操作流程（通用）**

##### Step A：檢視參考區參數
1. 點擊分類標籤（例如：「相機參數」）
2. 參考區會顯示來源料號的參數清單

##### Step B：選擇要複製的參數
有三種方式選擇參數：

**方式 1：手動勾選**
- 在參考區中，點擊參數前面的「☐」核取方塊
- 選中的參數會變成「☑」

**方式 2：使用快速選取按鈕**
```
[全選]    ← 勾選參考區中的所有參數
[全不選]  ← 取消所有勾選
[反選]    ← 勾選變不勾選，不勾選變勾選
```


##### Step C：複製參數到「已新增未修改區」
1. 勾選好要複製的參數後
2. 點擊「**複製到新增區 →**」按鈕
3. 參數會出現在中間的「已新增未修改區」
4. 參考區中該參數的背景會變成**淺綠色** 🟢

##### Step D：修改參數值

在「已新增未修改區」中修改參數：

1. **直接在表格中編輯**
   - 點擊參數的「值」欄位
   - 輸入新的數值
   - 按 Enter 確認

2. **數值輸入注意事項**
   - 整數參數：直接輸入數字（例如：`5000`）
   - 小數參數：使用小數點（例如：`3.14`）
   - 布林參數：輸入 `true` 或 `false`
   - 字串參數：直接輸入文字

3. **修改範圍限制**
   - 如果輸入超出範圍的值，系統會顯示錯誤訊息
   - 參考「參數說明」欄位，了解合理範圍

##### Step E：標記為已修改
修改完參數值後：

1. 勾選已修改的參數
2. 點擊「**標記為已修改 →**」按鈕
3. 參數會移動到「已新增已修改區」
4. 參考區中該參數的背景會變成**淺藍色** 🔵

##### Step F：儲存此分類
1. 確認「已新增已修改區」的參數都正確
2. 點擊視窗下方的「**儲存此分類**」按鈕
3. 系統會將參數寫入資料庫
4. 分類標籤會顯示 ✅ 圖示

#### **復原操作**

如果設定錯誤，可以復原：

**復原修改（從已修改→未修改）**
1. 在「已新增已修改區」勾選要復原的參數
2. 點擊「**← 復原修改**」按鈕
3. 參數會移回「已新增未修改區」，恢復原始值

**完全移除參數（從新增區→刪除）**
1. 在「已新增未修改區」或「已新增已修改區」勾選要移除的參數
2. 點擊「**← 移除選取項目**」按鈕
3. 參數會從新增區消失
4. 參考區中該參數的背景恢復成**白色** ⬜

---

### 📝 第四步：儲存並完成設定

#### **儲存方式選擇**

**方式 1：分次儲存（推薦）**
- 每完成一個分類就點擊「**儲存此分類**」
- **優點**：不會因為關閉視窗而遺失已設定的參數
- **適用**：設定複雜、需要長時間調整的情況

**方式 2：一次儲存全部**
- 等所有分類都設定完成
- 點擊「**儲存全部並關閉**」按鈕
- **優點**：一次完成所有操作
- **適用**：參數簡單、快速設定的情況

#### **儲存確認**

儲存時，系統會顯示確認對話框：

```
┌─────────────────────────────────┐
│  ⚠️ 確認儲存                    │
├─────────────────────────────────┤
│  即將儲存以下參數到資料庫：       │
│                                 │
│  目標料號：10215112360T          │
│  分類：相機參數                  │
│  參數數量：3                     │
│                                 │
│  - exposure = 6000              │
│  - gain = 12                    │
│  - delay = 150                  │
│                                 │
│  確定要儲存嗎？                  │
├─────────────────────────────────┤
│     [確定]        [取消]         │
└─────────────────────────────────┘
```

點擊「確定」後：
- ✅ 參數已成功儲存
- 📊 進度條更新
- 🎯 分類狀態圖示更新

---

### 📝 第五步：匯出工作階段（可選）

#### **什麼是工作階段？**
工作階段是一個 JSON 檔案，記錄了：
- 您選擇的來源料號
- 所有分類的參數設定
- 進度狀態

#### **為什麼要匯出？**
- 📄 **備份**：防止設定遺失
- 🔄 **快速復原**：下次可以直接載入
- 👥 **分享設定**：給其他工程師參考

#### **如何匯出？**
1. 點擊「**匯出工作階段**」按鈕
2. 選擇儲存位置
3. 檔案會自動命名為：`param_session_{料號}.json`
4. 預設位置：`bin\x64\Release\setting\`

#### **如何匯入？**
1. 開啟參數設定視窗
2. 點擊「**匯入工作階段**」按鈕
3. 選擇之前匯出的 JSON 檔案
4. 系統會自動載入所有設定

## 部分自動導入功能

### 🔄 什麼是「部分自動導入」？

**部分自動導入**是一個智能輔助功能，可以根據料號的基本資訊（如是否有油溝、PTFE顏色、尺寸等），**自動設定某些特定的參數**，節省您手動設定的時間。

### 🎯 功能特色

✅ **智能判斷**：根據料號特性自動決定參數值
✅ **快速設定**：一鍵完成多個條件式參數設定
✅ **減少錯誤**：避免人工設定錯誤
✅ **透明化**：顯示處理了哪些參數以及設定的理由

### 📋 會處理哪些參數？

「部分自動導入」主要處理以下類型的參數：

#### 1. **油溝相關參數（find_NROI）**

系統會根據是否有「內圈NROI模型」來自動設定：

| 條件 | 參數設定 | 說明 |
|------|---------|------|
| **有內圈NROI模型** | `find_NROI = 1` | 啟用非ROI區域檢測 |
|  | `expandNROI_in = 15` | 內側擴展15像素 |
|  | `expandNROI_out = 30` | 外側擴展30像素 |
| **無內圈NROI模型** | `find_NROI = 0` | 停用非ROI區域檢測 |
|  | `expandNROI_in = 0` | 不擴展 |
|  | `expandNROI_out = 0` | 不擴展 |

#### 2. **其他條件式參數**

系統會根據料號資訊（如ID、OD、H、PTFE顏色等）自動設定其他相關參數。

### 🚀 如何使用

#### **Step 1：確認料號基本資訊已建立**

在使用「部分自動導入」之前，請確認：
- [ ] 目標料號已在系統中建立
- [ ] 料號的基本資訊（ID、OD、H、是否有油溝、PTFE顏色等）已正確填寫

#### **Step 2：開啟參數設定視窗**

1. 打開新料號的「參數設定」視窗
2. 選擇來源料號（參考範本）

#### **Step 3：點擊「部分自動導入」按鈕**

在任何分類標籤（相機參數、位置參數、檢測參數、時間參數）的右上角，您會看到：

```
[🔄 部分自動導入]
```

點擊此按鈕。

#### **Step 4：確認執行**

系統會顯示確認對話框：

```
┌─────────────────────────────────────────┐
│  ⚠️ 確認部分自動導入                     │
├─────────────────────────────────────────┤
│  即將對料號 10215112360T 執行部分自動     │
│  導入                                   │
│                                         │
│  將會處理以下條件式參數：                 │
│  • find_NROI：根據油溝狀態自動設定        │
│  • 其他特殊參數依據料號特性調整           │
│                                         │
│  確定要執行嗎？                          │
├─────────────────────────────────────────┤
│     [是(Y)]        [否(N)]              │
└─────────────────────────────────────────┘
```

點擊「**是(Y)**」確認執行。

#### **Step 5：檢視處理結果**

處理完成後，系統會顯示結果：

```
┌─────────────────────────────────────────┐
│  ✅ 部分自動導入完成                     │
├─────────────────────────────────────────┤
│  處理了 6 個參數：                       │
│                                         │
│  • find_NROI_1 = 1 (InnerModel=true)   │
│  • expandNROI_in_1 = 15                │
│  • expandNROI_out_1 = 30               │
│  • find_NROI_2 = 1 (InnerModel=true)   │
│  • expandNROI_in_2 = 15                │
│  • expandNROI_out_2 = 30               │
│                                         │
├─────────────────────────────────────────┤
│     [確定]                              │
└─────────────────────────────────────────┘
```

#### **Step 6：檢查並儲存**

1. 自動導入的參數會直接出現在「**已新增已修改區**」
2. 您可以在該區域檢查這些參數是否正確
3. 如果需要微調，可以直接編輯參數值
4. 確認無誤後，點擊「**儲存此分類**」

### ⚠️ 重要注意事項

1. **不會覆蓋已設定的參數**
   - 「部分自動導入」只會處理尚未設定的參數
   - 如果參數已經在「已新增」區域，不會被覆蓋

2. **需要料號基本資訊**
   - 功能依賴料號的基本資訊（ID、OD、H、是否有油溝等）
   - 如果資訊不完整，某些參數可能無法自動設定

3. **仍需人工確認**
   - 自動導入的參數僅供參考
   - 建議您仍需檢查並確認這些參數是否符合實際需求

4. **補充性質功能**
   - 這個功能是用來**輔助**您快速設定參數
   - 並不能取代完整的參數設定流程
   - 仍需要手動設定其他大部分參數

### 💡 使用建議

**適合使用「部分自動導入」的情況：**
- ✅ 新料號與舊料號非常相似（只有尺寸或顏色微調）
- ✅ 您對系統的參數邏輯已經很熟悉
- ✅ 想要快速設定條件式參數（如油溝相關）

**不適合使用的情況：**
- ❌ 新料號與舊料號差異很大
- ❌ 您還不熟悉各個參數的意義
- ❌ 料號基本資訊不完整或不確定

**最佳實踐：**
```
1. 先手動複製並設定相機參數
2. 完成位置參數校準
3. 點擊「部分自動導入」快速設定條件式參數
4. 手動微調檢測參數
5. 設定時間參數
6. 完整測試驗證
```
