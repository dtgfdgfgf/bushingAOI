# 白色像素校正工具使用說明

## 功能用途

白色像素校正工具用於設定白色像素占比的檢測參數,透過分析正常樣品和誤觸樣品的原始圖像,自動計算並推薦四個關鍵參數,作為產品白色區域檢測的判定標準。

---

## 核心概念

### 四個參數的意義

本工具會計算並套用以下四個參數:

1. **white_{站點編號}**: 正常樣品的白色像素占比基準值(中位數)
2. **whiteNULL_{站點編號}**: 誤觸樣品的白色像素占比基準值(中位數)
3. **whiteTolerance_{站點編號}**: 容忍度,定義正常樣品的可接受偏差範圍
4. **whiteThresh_{站點編號}**: 二值化閾值,影響白色像素的判定標準(0-255)

### 雙組分析機制

系統需要同時分析兩組照片:
- **正常樣品組**: 合格產品的照片,用於計算正常基準值
- **誤觸樣品組**: 不合格產品(如污點、髒污)的照片,用於計算異常基準值

透過雙組分析,系統可以:
- 計算兩組樣品的區分度
- 檢測判定區間是否重疊
- 確保設定的參數能有效區分合格與不合格產品

## 操作步驟

### Step 1: 開啟工具並選擇站點

1. 在參數設定視窗中,點擊「檢測參數」標籤
2. 點擊「白色像素校正」按鈕
3. **選擇站點**: 在下拉選單中選擇站點1-4

**重要**: 每個站點都有獨立的參數設定,請確認選擇正確的站點

---

### Step 2: 選擇正常樣品照片

1. 點擊「選擇正常照片」按鈕
2. 選擇 **3張以上** 正常產品的原始圖像
3. 建議選擇不同時間拍攝的照片(測試穩定性)

**圖像要求**:
- 使用原始圖像(相機拍攝的完整影像,非 ROI 裁切圖)
- 照片應為合格產品
- 避免過度曝光或過暗
- 產品應清潔、無污漬

---

### Step 3: 選擇誤觸樣品照片

1. 點擊「選擇誤觸照片」按鈕
2. 選擇 **10張以上** 不合格產品的原始圖像
3. 建議涵蓋各種常見瑕疵類型

**誤觸樣品定義**:
- 有明顯污點、髒污的產品
- 顏色異常的產品
- 其他應被判定為不良的產品

**重要**: 誤觸樣品的數量和代表性直接影響系統的區分能力

---

### Step 4: 調整二值化閾值(可選)

1. 使用二值化閾值滑桿調整 **whiteThresh** 值(預設 180)
2. 閾值範圍: 0-255
3. 閾值越高,判定為白色的標準越嚴格

**調整建議**:
- 產品白色區域偏暗: 降低閾值(如 160-170)
- 產品白色區域偏亮: 提高閾值(如 190-200)
- 若不確定,建議先使用預設值 180 進行分析

**注意**: 調整閾值後需要重新分析才會生效

---

### Step 5: 開始分析

1. 確認已選擇正常樣品和誤觸樣品照片
2. 點擊「開始分析」按鈕
3. 系統會依序處理所有照片:
   - 轉換為灰階圖像
   - 進行二值化處理
   - 計算白色像素占比
   - 標記離群值樣品
   - 計算統計數據

分析過程中會顯示進度條和處理狀態

---

### Step 6: 檢視分析結果

分析完成後,系統會顯示以下資訊:

#### 圖片列表
顯示所有照片的分析結果,包含:
- 樣品類型(正常/誤觸)
- 照片名稱
- 白色像素占比(%)
- 狀態標記:
  - **正常**: 黑底白字
  - **1σ異常**: 淡藍底黑字
  - **2σ異常**: 淡橙底黑字
  - **組內離群**: 淡黃底深橙字(該樣品在其組內偏離較大)
  - **疑似誤放**: 紅底白字加粗(可能選錯組別)

#### 統計分析結果面板

**正常樣品統計**(藍色文字):
- 正常樣品數/有效樣品
- 平均占比/中位數占比
- 標準差(穩定性指標)
- 最小值/最大值
- 1σ異常/2σ異常數量
- 推薦值(white 參數)

**誤觸樣品統計**(紅色文字):
- 誤觸樣品數/有效樣品
- 平均占比/中位數占比
- 標準差
- 最小值/最大值
- 1σ異常/2σ異常數量
- 推薦值(whiteNULL 參數)

#### 折線圖
視覺化顯示所有照片的白色像素占比趨勢:
- 藍線: 正常樣品
- 紅線: 誤觸樣品
- 綠線: 平均值
- 黃線: ±1σ 範圍
- 橙線: ±2σ 範圍

#### 推薦參數
顯示計算出的四個參數值:
- white_{站點} = X.X%
- whiteNULL_{站點} = X.X%
- whiteTolerance_{站點} = X.X%
- whiteThresh_{站點} = XXX

#### 區分度資訊
系統會自動計算正常樣品和誤觸樣品的判定區間,並顯示:
- **安全距離**: 兩組區間不重疊,顯示安全距離(綠色,良好)
- **重疊警告**: 兩組區間重疊,顯示重疊範圍(紅色,需調整)

---

### Step 7: 預覽照片細節

#### 正常樣品預覽(左側藍框)
- **原圖標註**: 顯示原始圖像及分析資訊
- **二值化圖**: 顯示二值化處理後的黑白圖像
- 使用「上一張」/「下一張」按鈕切換照片
- 點擊「全螢幕」可放大查看

#### 誤觸樣品預覽(右側紅框)
- 與正常樣品預覽功能相同
- 可獨立瀏覽誤觸樣品照片

**提示**: 點擊圖片列表中的照片,會自動切換到對應的預覽

---

### Step 8: 判斷結果品質

#### 情況 A: 區分度良好

```
安全距離：6.5%

容忍度計算：

統計基準：1.50% (正常樣品 3σ)

最終容忍度：1.50% (取最大值)

推薦操作：
✓ 點擊「套用推薦值」→ 儲存參數
```

**說明**: 兩組樣品有明顯區隔,設定的參數可以有效區分合格與不合格產品

---

#### 情況 B: 區間重疊(需調整)

```
警告：正常與誤觸樣品的判定區間重疊 2.5%

正常判定區間：[9.0%, 14.0%]
誤觸判定區間：[11.5%, 16.5%]
重疊區間：[11.5%, 14.0%]

容忍度計算：
距離基準：0.75% (兩組距離的 1/3)
統計基準：1.20% (正常樣品 3σ)
經驗基準：1.00%
最終容忍度：1.20% (取最大值)

推薦操作：
1. 調整 whiteThresh 閾值(嘗試提高或降低 10-20)
2. 重新分析查看區分度
3. 若仍重疊,檢查照片選擇是否正確
4. 聯絡工程師協助優化
```

**說明**: 兩組樣品的判定區間有重疊,可能導致誤判,需要調整參數或檢查照片品質

---

#### 情況 C: 疑似誤放樣品

```
⚠️ 警告提示標籤顯示：
疑似問題照片：正常樣品 #3、#7（疑似誤放） / 誤觸樣品 #2（組內離群）

推薦操作：
1. 點擊圖片列表中標記為「疑似誤放」的照片
2. 查看預覽圖像確認是否選錯組別
3. 移除有問題的照片
4. 點擊「清除分析結果」
5. 重新選擇照片並分析
```

**說明**: 系統偵測到某些照片的數值與其所屬組別不符,可能是選錯組別

---

### Step 9: 套用參數

**如果區分度良好**:
1. 點擊「套用推薦值」按鈕
2. 系統會更新四個參數到資料庫:
   - white_{站點}
   - whiteNULL_{站點}
   - whiteTolerance_{站點}
   - whiteThresh_{站點}
3. 參數會自動新增到「檢測參數」的「已新增已修改區」
4. 確認訊息會顯示套用的參數值和判定區間

**如果區間重疊**:
1. **不要直接套用**
2. 調整 whiteThresh 閾值
3. 點擊「開始分析」重新分析
4. 重複步驟直到區分度良好
5. 若調整多次仍無法改善,聯絡工程師檢查系統或照片品質

## 注意事項

### 1. 照片選擇原則

**正常樣品**:
- ✓ 合格產品的照片
- ✓ 照明正常、產品清潔
- ✓ 不同時間拍攝(間隔幾分鐘或幾小時)
- ✓ 數量: 3張以上(建議 5-10 張)
- ✗ 避免過度曝光或過暗
- ✗ 避免有污漬或瑕疵的產品

**誤觸樣品**:
- ✓ 不合格產品的照片(污點、髒污、顏色異常)
- ✓ 涵蓋各種常見瑕疵類型
- ✓ 數量: 10張以上(建議 15-20 張)
- ✗ 避免選擇合格產品

### 2. 統計指標的意義

**標準差**(穩定性指標):
- < 0.5%: 非常穩定 ✓
- 0.5-1.0%: 可接受,建議檢查
- \> 1.0%: 不穩定,必須檢查 ✗

**1σ異常**: 樣品占比偏離平均值 1 個標準差以上(輕微偏離)
**2σ異常**: 樣品占比偏離平均值 2 個標準差以上(明顯偏離)

### 3. 離群值檢測機制

系統會自動執行三層離群值檢測:

**第一層: σ 檢測**(統計偏離)
- 檢測樣品是否偏離組內平均值 1σ 或 2σ 以上

**第二層: IQR 檢測**(組內離群)
- 使用四分位距(IQR)檢測組內極端值
- 超出 [Q1 - 1.5×IQR, Q3 + 1.5×IQR] 範圍視為組內離群

**第三層: 組間交叉檢測**(疑似誤放)
- 檢測樣品是否更接近另一組的中位數
- 標記為「疑似誤放」表示可能選錯組別

### 4. 影響穩定性的因素

- **照明穩定性**(最主要因素): 確認照明電源穩定、燈管未老化
- **相機設定**: 檢查相機白平衡、曝光時間、增益
- **產品一致性**: PTFE 顏色批次差異
- **環境光干擾**: 避免外部光源干擾、加裝遮光罩

### 5. 使用原始圖像

- **本工具使用完整的原始圖像**,不進行 ROI 裁切
- 與其他校正工具(如對比檢測、物體偏移)使用 ROI 圖像不同
- 這樣可以分析整個產品區域的白色像素分布

## 使用技巧

### 改善區分度的方法

如果正常樣品和誤觸樣品的判定區間重疊,可以嘗試:

#### 方法 1: 調整二值化閾值
```
1. 嘗試提高閾值(如 180 → 200)
   - 適用於產品白色區域偏亮的情況
   - 可以降低白色像素占比,拉開兩組距離

2. 嘗試降低閾值(如 180 → 160)
   - 適用於產品白色區域偏暗的情況
   - 可以提高白色像素占比,拉開兩組距離

3. 每次調整 10-20,重新分析查看效果
```

#### 方法 2: 檢查照片品質
```
1. 確認正常樣品都是合格產品
2. 確認誤觸樣品都有明顯瑕疵
3. 移除疑似誤放的照片
4. 增加照片數量提高統計準確性
```

#### 方法 3: 優化拍攝條件
```
1. 檢查照明穩定性
   - 確認照明電源穩定
   - 檢查燈管是否老化
   - 調整照明角度避免反光

2. 調整相機參數
   - 檢查相機白平衡設定
   - 調整曝光時間避免過曝
   - 降低增益減少雜訊

3. 環境控制
   - 避免外部光源干擾
   - 加裝遮光罩
   - 確保工作區域照明穩定
```

## 常見問題

### Q1: 什麼是白色像素占比?

**答案**:
白色像素占比 = (白色像素數量 / 總像素數量) × 100%

系統會:
1. 將原始圖像轉換為灰階
2. 使用 whiteThresh 閾值進行二值化(灰度 ≥ 閾值視為白色)
3. 計算白色像素占整張圖像的百分比

這個比例反映了產品的白色區域大小,可用於檢測污點、髒污等瑕疵。

---

### Q2: 為什麼需要同時選擇正常樣品和誤觸樣品?

**答案**:
雙組分析的優點:
- **計算容忍度**: 根據兩組的距離和分布,自動計算合理的容忍度
- **檢測區分度**: 確保設定的參數能有效區分合格與不合格產品
- **避免誤判**: 防止參數設定過寬(漏判)或過窄(誤判)
- **優化閾值**: 透過調整 whiteThresh 最大化兩組的區分度

單組分析無法確認參數是否能有效排除不良品。

---

### Q3: 推薦的照片數量是多少?

**答案**:
- **正常樣品**: 3張以上(建議 5-10 張)
  - 至少 3 張才能計算統計指標
  - 5-10 張可以更準確評估穩定性

- **誤觸樣品**: 10張以上(建議 15-20 張)
  - 瑕疵類型多樣,需要更多樣品
  - 數量越多,統計越準確

照片過少會導致統計不準確,過多則增加分析時間但改善有限。

---

### Q4: 什麼情況下需要重新校正?

**答案**:
通常不需要經常重新校正,除非:
- 更換了照明設備
- 更換了相機
- 調整了相機參數(曝光、增益、白平衡)
- PTFE 供應商改變(顏色有差異)
- 檢測穩定性下降(誤判率上升)

建議每 3-6 個月檢查一次穩定性,或在系統環境變更後重新校正。

---

### Q5: 如何處理「疑似誤放」的照片?

**答案**:
1. 點擊圖片列表中標記為「疑似誤放」的照片
2. 查看預覽圖像:
   - 如果確實選錯組別: 移除該照片,重新選擇正確的組別
   - 如果確認沒有選錯: 該照片可能是特殊案例,可保留或移除
3. 點擊「清除分析結果」按鈕
4. 重新選擇照片並分析

---

### Q6: 為什麼調整閾值後需要重新分析?

**答案**:
二值化閾值(whiteThresh)直接影響白色像素的判定標準:
- 閾值改變 → 白色像素數量改變 → 白色像素占比改變
- 所有統計數據(平均、中位數、標準差)都會改變
- 推薦參數(white、whiteNULL、whiteTolerance)都會改變

因此調整閾值後必須重新分析才能得到正確的結果。

---

### Q7: 容忍度(whiteTolerance)是如何計算的?

**答案**:
系統使用混合策略,結合三種方法計算,取最大值:

1. **距離基準**: (正常中位數 - 誤觸中位數) / 3
   - 基於兩組的距離,確保有足夠的安全距離

2. **統計基準**: 正常樣品 3σ
   - 基於正常樣品的穩定性,容納大部分正常樣品

3. **經驗基準**: 1.0%
   - 最低容忍度,避免過度嚴格

取最大值可確保容忍度:
- 足夠容納正常樣品的偏離
- 保持與誤觸樣品的安全距離
- 不會過度嚴格導致誤判

---

### Q8: 圖片列表中的背景顏色代表什麼?

**答案**:
- **黑底白字**: 正常樣品,無異常
- **淡藍底黑字**: 1σ 異常(輕微偏離)
- **淡橙底黑字**: 2σ 異常(明顯偏離)
- **淡黃底深橙字**: 組內離群(該樣品在其組內偏離較大)
- **紅底白字加粗**: 疑似誤放(可能選錯組別)

這些標記幫助快速識別需要檢查的照片。

---

### Q9: 清除分析結果會刪除照片嗎?

**答案**:
不會。「清除分析結果」只會:
- 清除分析數據(白色像素占比、統計資訊)
- 清除預覽圖像
- 清除推薦參數

原始照片檔案不會被刪除或修改。您可以重新選擇照片並分析。

## 介面說明

### 控制按鈕

- **選擇正常照片**: 選擇合格產品的原始圖像
- **選擇誤觸照片**: 選擇不合格產品的原始圖像
- **開始分析**: 執行白色像素占比分析
- **套用推薦值**: 將計算出的四個參數儲存到資料庫
- **清除分析結果**: 清除目前的分析數據(不刪除原始照片)
- **上一張/下一張**: 切換預覽照片
- **全螢幕**: 放大查看當前預覽圖像

### 二值化閾值滑桿

- 範圍: 0-255
- 預設: 180
- 調整後需重新分析才會生效
- 閾值越高,判定為白色的標準越嚴格

## 工作流程總結

```
1. 開啟工具 → 選擇站點
   ↓
2. 選擇正常樣品照片(3張以上)
   ↓
3. 選擇誤觸樣品照片(10張以上)
   ↓
4. 調整二值化閾值(可選,預設 180)
   ↓
5. 開始分析
   ↓
6. 檢視分析結果:
   - 圖片列表(檢查離群值/疑似誤放)
   - 統計分析(檢查穩定性)
   - 折線圖(視覺化趨勢)
   - 推薦參數
   - 區分度資訊
   ↓
7. 判斷結果品質:

   區分度良好?
   ├─ 是 → 套用推薦值 → 完成
   └─ 否 → 調整閾值/檢查照片 → 回到步驟 5
```
