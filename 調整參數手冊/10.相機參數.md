## 參數說明

相機參數決定了系統如何拍攝產品影像，直接影響後續所有檢測步驟的準確性。

### 📊 參數列表

| 參數名稱 | 說明 | 單位 | 典型範圍 | 預設值參考 |
|---------|------|------|---------|-----------|
| `exposure` | 曝光時間 | 微秒 (μs) | 1000-10000 | 5000 |
| `gain` | 增益值 | - | 0-20 | 10 |
| `delay` | 拍照延遲 | 毫秒 (ms) | 50-500 | 100 |

### 🔍 參數詳解

#### **1. exposure（曝光時間）**

**作用**：控制相機感光元件曝光的時間長度

**影響**：
- 曝光時間越長 → 影像越亮，但可能產生運動模糊
- 曝光時間越短 → 影像越暗，但能凍結快速移動的物體

**調整原則**：
```
影像太暗   → exposure 增加 (50-200)
影像太亮   → exposure 減少 (50-200)
影像模糊   → exposure 減少（避免運動模糊）
產品移動慢 → 可以使用較長的曝光時間
產品移動快 → 必須使用較短的曝光時間
```

---

#### **2. gain（增益值）**

**作用**：電子放大影像訊號，類似相機的ISO

**影響**：
- 增益值越高 → 影像越亮，但雜訊（噪點）增加
- 增益值越低 → 影像雜訊少，但可能太暗

**調整原則**：
```
在曝光值調整完後再調整增益
雜訊太多 → gain 減少 1-2
亮度不足 → gain 增加 1-2

⚠️ 注意：gain 是用來微調亮度，不要當作主要調整手段
```

**典型值範圍**：
- 理想照明環境：5-10
- 一般照明環境：10-15
- 較暗環境：15-20
- ❌ 避免：> 20（雜訊會非常明顯）

- 請自行依照實際影像斟酌exposure與gain的增減
---

#### **3. delay（拍照延遲）**

**作用**：PLC觸發訊號到相機實際拍照的等待時間

**影響**：
- 延遲太短 → 產品還沒移動到位就拍照
- 延遲太長 → 產品已經移動過頭才拍照
- 延遲適當 → 產品剛好在檢測位置時拍照

**調整原則**：
```
延遲時間 = PLC觸發 → 產品移動到檢測位置的時間

考慮因素：
1. 轉盤速度
2. 觸發點到檢測位置的距離

```
小建議：在本機台的轉速之下，1 像素的距離大約等於1 ms的時間
舉例：假設您認為影像中的樣品要往左或右平移 100 像素，就將delay增加或減少 100 ms

## 完整設定流程

### 📝 Step 1：複製參考參數

1. 點擊「**相機參數**」標籤
2. 在參考區勾選所有相機參數：
   - ☑ `exposure`
   - ☑ `gain`
   - ☑ `delay`
3. 點擊「**複製到新增區 →**」按鈕

**結果**：
```
參考區（來源料號）             已新增未修改區
┌──────────────────┐           ┌──────────────────┐
│☐ exposure = 5000 │ 🟢  →     │ exposure = 5000  │
│☐ gain     = 10   │ 🟢  →     │ gain     = 10    │
│☐ delay    = 100  │ 🟢  →     │ delay    = 100   │
└──────────────────┘           └──────────────────┘
```

---

### 📝 Step 2：調整曝光值（exposure）

#### **方法一：直接測試調整（推薦）**

1. **先測試來源料號的值**
   - 在主視窗中手動觸發拍照
   - 或放置樣品等待PLC自動觸發

2. **觀察影像效果**
   ```
   影像太暗 → exposure 增加 
   影像太亮 → exposure 減少 
   影像模糊 → exposure 減少 
   影像完美 → 不需調整
   ```

3. **在「已新增未修改區」編輯參數**
   - 點擊 exposure 的值欄位
   - 輸入新數值
   - 按 Enter 確認

4. **重複測試調整**
   - 儲存參數（臨時）
   - 再次拍照測試
   - 直到影像亮度適中

#### **方法二：根據產品特性估算**

根據目標料號與參考料號的差異：

```
目標料號更大、顏色更深   → exposure 增加
目標料號更小、顏色更淺   → exposure 減少
尺寸、顏色相似           → 不需調整
```

---

### 📝 Step 3：調整增益值（gain）

⚠️ **重要**：一定要先調整好 exposure，再調整 gain！

1. **觀察影像雜訊**
   - 放大影像查看細節
   - 特別注意均勻區域（如背景、產品表面）

2. **判斷雜訊程度**
   ```
   雜訊很多（顆粒感明顯） → gain 減少 2-5
   雜訊適中（輕微顆粒）   → gain 減少 1-2
   雜訊很少但太暗         → gain 增加 1-2
   雜訊少且亮度適中       → 不需調整
   ```

3. **在「已新增未修改區」編輯參數**
   - 點擊 gain 的值欄位
   - 輸入新數值（例如：`8`）
   - 按 Enter 確認

4. **重複測試調整**
   - 儲存參數（臨時）
   - 再次拍照測試
   - 確認雜訊和亮度都在可接受範圍

#### **增益調整原則**

```
優先順序：

1️⃣ 先用 exposure 調整亮度（主要手段）
2️⃣ 再用 gain 微調（輔助手段）
3️⃣ 寧可影像稍暗，也不要 gain 太高導致雜訊多

理由：
exposure 提高亮度 → 不增加雜訊
gain 提高亮度     → 會增加雜訊
```

---

### 📝 Step 4：調整拍照延遲（delay）

#### **方法一：實測法（最準確）**

1. **使用來源料號的 delay 值先測試**
   - 讓產品通過檢測站
   - 觀察拍照時產品的位置

2. **判斷延遲是否適當**
   ```
   產品還沒到檢測位置 → delay 太短，需增加
   產品剛好在中心     → delay 正確，不需調整
   產品已經過頭       → delay 太長，需減少
   ```

3. **調整步驟**
   ```
小建議：在本機台的轉速之下，1 像素的距離大約等於1 ms的時間
舉例：假設您認為影像中的樣品要往左或右平移 100 像素，就將delay增加或減少 100 ms
補充：機台轉速固定不可變動
   ```

4. **重複測試**
   - 修改 delay 值
   - 儲存參數
   - 再次測試
   - 直到產品位置正確


---

### 📝 Step 5：測試拍照效果

完成參數調整後，進行綜合測試：

#### **測試項目**

1. **影像亮度**
   ```
   ✅ 產品輪廓清晰可見
   ✅ 產品表面細節清楚
   ✅ 不會過亮（沒有大片白色區域）
   ✅ 不會過暗（不會看不清細節）
   ```

2. **影像雜訊**
   ```
   ✅ 放大影像後，雜訊不明顯
   ✅ 均勻區域沒有明顯顆粒感
   ✅ 邊緣清晰，不會被雜訊干擾
   ```

3. **影像清晰度**
   ```
   ✅ 產品邊緣銳利，沒有模糊
   ✅ 文字或標記清晰可讀
   ✅ 沒有運動模糊的拖影
   ```

4. **產品位置**
   ```
   ✅ 產品在影像中心（或預定位置）
   ✅ 產品完整，沒有被切掉
   ✅ 每次拍照位置一致
   ```

#### **測試方法**

**快速測試（5次）**
```
1. 連續放置 5 個樣品
2. 觀察 5 張影像
3. 確認一致性
```

**完整測試（20次）**
```
1. 連續放置 20 個樣品
2. 儲存所有影像
3. 逐一檢查
4. 記錄異常影像數量
5. 目標：異常率 < 5% (1/20)
```

---

### 📝 Step 6：標記並儲存

確認參數都正確後：

1. **勾選所有已調整的參數**
   - ☑ exposure（如果有修改）
   - ☑ gain（如果有修改）
   - ☑ delay（如果有修改）

2. **點擊「標記為已修改 →」**
   - 參數會移動到「已新增已修改區」
   - 參考區背景變成淺藍色 🔵

3. **點擊「儲存此分類」**
   - 系統顯示確認對話框
   - 點擊「確定」
   - 參數寫入資料庫

4. **確認儲存成功**
   - 狀態列顯示 ✅ 相機(3/3)
   - 進度條更新

## 常見問題

### ❓ Q1：調整了 exposure 但影像還是太暗怎麼辦？

**可能原因**：

1. **照明不足**
   ```
   解決方法：
   - 調整gain，或繼續調exposure
   ```

2. **產品本身顏色很深**
   ```
   解決方法：
   - exposure 可以調更高
   - 搭配 gain 微調
   ```

3. **相機設定問題**
   ```
   解決方法：
   - 檢查相機硬體連線
   - 確認相機型號設定正確
   - 聯絡工程師
   ```

   備註：❌請勿自行更改電源電壓

---

### ❓ Q2：影像清晰但總是偏左或偏右？

**可能原因**：這是 delay 的問題，不是 exposure/gain

```
解決方法：
1. 產品偏左（還沒到位） → delay 增加
2. 產品偏右（已經過頭） → delay 減少
3. 每次調整 10-50 ms
4. 重複測試直到居中
```

---

### ❓ Q3：影像一下亮一下暗，不穩定？

**可能原因**：

1. **照明不穩定**
   ```
   解決方法：
   - 檢查照明電源
   - 檢查照明設備是否故障
   - 固定照明位置
   ```

2. **產品反光**
   ```
   解決方法：
   - 降低 exposure
   - 降低 gain
   ```

3. **環境光干擾**
   ```
   解決方法：
   - 加裝遮光罩
   - 使用專用照明區域
   ```

---

### ❓ Q4：新料號和舊料號很像，參數可以直接複製嗎？

**答案**：可以，但仍需測試確認

```
建議流程：
1. 直接複製參考料號的參數
2. 不要修改，直接標記為已修改
3. 儲存到資料庫
4. 實際拍照測試
5. 如果效果不理想，再回來微調
```

**適用情況**：
- ✅ 同系列產品，只是尺寸微調
- ✅ 同顏色、同材質
- ✅ 生產線速度相同

**不適用情況**：
- ❌ 顏色差異大（如深色→淺色）
- ❌ 尺寸差異大（如大型→小型）

---

### ❓ Q5：儲存後測試發現參數不對，怎麼改？

**方法一：重新開啟參數設定視窗**
```
1. 關閉目前的參數設定視窗
2. 重新開啟目標料號的參數設定
3. 選擇相同的來源料號
4. 修改參數
5. 重新儲存
```

**方法二：直接在主視窗修改（不建議）**
```
⚠️ 只在緊急情況使用
- 直接在資料庫編輯（需要權限）
- 風險高，容易出錯
```

## 快速參考卡

列印此卡片放在操作台旁：

```
┌────────────────────────────────────────────┐
│ 相機參數快速參考                            │
├────────────────────────────────────────────┤
│ 症狀              │ 調整            │ 幅度 │
├──────────────────┼────────────────┼──────┤
│ 影像太暗          │ exposure ↑     │ +500~1000│
│ 影像太亮          │ exposure ↓     │ -500~1000│
│ 影像模糊          │ exposure ↓     │ -200~500 │
│ 雜訊太多          │ gain ↓         │ -2~5     │
│ 產品還沒到位      │ delay ↑        │ +10~50  │
│ 產品已經過頭      │ delay ↓        │ -10~50  │
└────────────────────────────────────────────┘

調整優先順序：
1️⃣ exposure（主要調整亮度）
2️⃣ gain（微調亮度、控制雜訊）
3️⃣ delay（調整拍照時機）

測試標準：
✅ 亮度適中、無過曝或過暗
✅ 雜訊少、影像乾淨
✅ 清晰無模糊
✅ 產品位置正確
```
