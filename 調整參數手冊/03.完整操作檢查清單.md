# 檢測參數設定 - 完整操作檢查清單

**列印此清單，每次設定或調整料號參數時逐項勾選確認**

---

## 📋 前置作業

- [ ] 確認目標料號已在系統中建立（新料號需先建立）
- [ ] 選擇合適的參考料號（產品外型、尺寸接近、PTFE同色，可選擇自己）
- [ ] 準備測試樣品（良品×10、不良品×5）
- [ ] AI 模型檔案已放置到 models 資料夾（新料號需要）
- [ ] 設備狀態正常（相機、PLC、照明）

---

## 1️⃣ 相機參數

- [ ] 複製參考料號的相機參數
- [ ] 調整 exposure（測試拍照效果）
- [ ] 調整 gain（檢查雜訊）
- [ ] 調整 delay（考慮產品大小）
- [ ] 測試拍照，確認影像品質
- [ ] 標記為已修改
- [ ] 儲存此分類

---

## 2️⃣ 位置參數

- [ ] 開啟圓形校準工具
- [ ] 拍攝或載入測試影像
- [ ] 標記所有圓形區域
- [ ] 確認圓心和半徑正確
- [ ] 儲存校準結果
- [ ] 回到位置參數標籤，確認參數已載入
- [ ] 儲存此分類

---

## 3️⃣ 檢測參數

- [ ] 複製參考料號的檢測參數
- [ ] 點擊「部分自動導入」快速設定條件式參數
- [ ] 使用「對比檢測校正」工具
- [ ] 使用「檢測經度校正」工具
- [ ] 使用「白色像素校正」工具
- [ ] 使用「開口閾值校正」工具
- [ ] 使用「物體偏移校正」工具
- [ ] 標記為已修改
- [ ] 儲存此分類

---

## 4️⃣ 時間參數

- [ ] 複製參考料號的時間參數
- [ ] 點擊「根據OD自動計算時間參數」
- [ ] 實測驗證時間參數
- [ ] 根據實際情況調整（推早了或推晚了）
- [ ] 標記為已修改
- [ ] 儲存此分類

---

## 5️⃣ 完成

- [ ] 匯出工作階段（備份）
- [ ] 點擊「儲存全部並關閉」
- [ ] 在主視窗中測試完整流程
- [ ] 確認檢測結果正確
- [ ] 通知相關人員參數已更新（新料號可開始使用）
- [ ] 記錄在料號管理表中

---

## ✅ 最終確認

### 影像品質
- [ ] 影像亮度適中
- [ ] 無過度雜訊
- [ ] 清晰無模糊
- [ ] 產品位置正確

### 檢測功能
- [ ] 良品能正確通過
- [ ] 不良品能正確偵測
- [ ] 無誤判或漏檢

### 系統運作
- [ ] PLC 通訊正常
- [ ] 推料時機準確
- [ ] 資料儲存正常
