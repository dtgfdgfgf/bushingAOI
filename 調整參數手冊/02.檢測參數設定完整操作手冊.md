## 什麼是檢測參數設定

### 🎯 功能目的
無論是要檢測**新的產品型號**（新料號）或**調整現有產品的檢測參數**（舊料號），都需要告訴 AOI 檢測系統：
- 相機要怎麼拍照（曝光、增益）
- 產品在影像中的位置在哪裡
- 要用什麼標準來判斷產品合格或不合格
- 檢測的時間要多長

這個功能就是讓您設定或修改這些參數的地方，**適用於新料號設定與舊料號參數調整**。

### 🔄 為什麼要「參考其他料號」
設定參數時，我們可以參考已存在的料號設定：
1. 選一個**相似的料號**當作「參考範本」（可以是其他料號，也可以是自己）
2. **複製**它的參數到「已添加區」
3. 只**修改**需要調整的地方
4. **儲存**成目標料號的參數

**特殊情況：舊料號微調**
- 如果只是要微調現有料號的參數，可以選擇**自己作為參考料號**
- 系統會將當前參數複製到「已添加區」，您只需修改要調整的部分即可

這樣比從零開始設定快很多！

## 開始之前的準備

### ✅ 必備資料清單

在開始設定之前，請先準備好以下資料：

#### 1. **目標料號基本資訊**
- [ ] 目標料號編號（例如：`10215112360T`）
- [ ] 目標料號名稱（例如：「大型襯套V2」）
- [ ] 產品規格書或技術圖面（如為新料號）

#### 2. **參考料號選擇**
- [ ] 找一個**最相似**的料號當參考（產品外型、尺寸接近、PTFE面同色）
- [ ] 如果是微調現有料號，可選擇**自己作為參考料號**
- [ ] 確認參考料號的參數在系統中**運作正常**

#### 3. **測試樣品**
- [ ] 準備 **5-10 個良品樣品**（用於測試和校準）
- [ ] 準備 **2-3 個不良品樣品**（用於驗證缺陷檢測）

#### 4. **設備狀態**
- [ ] 確認相機已連接並正常運作
- [ ] 確認 PLC 連線正常
- [ ] 確認照明設備正常

#### 5. **AI 模型檔案（如果需要）**
- [ ] 準備好目標料號的 TensorRT 模型檔案（`.pt` 檔案，如為新料號）
- [ ] 模型檔案命名規則：`{料號}_{in|out|1|2}.pt`
- [ ] 放置位置：`bin\x64\Release\models\` 資料夾

```

### 📊 三區域架構詳解

#### **區域 1：參考區（來源料號）**
- **位置**：最左邊
- **顏色標示**：
  - ⬜ **白色背景**：這個參數還沒被複製
  - 🟢 **淺綠色背景**：這個參數已被複製到「已新增未修改區」
  - 🔵 **淺藍色背景**：這個參數已被複製且已修改
- **功能**：顯示舊料號的所有參數，供您參考和複製
- **特性**：**唯讀**，不能修改，也不會因為您的操作而消失

#### **區域 2：已新增未修改區**
- **位置**：中間
- **功能**：放置從參考區複製過來、**但還沒修改**的參數
- **操作**：您可以直接在這裡編輯參數值

#### **區域 3：已新增已修改區**
- **位置**：最右邊
- **功能**：放置**已經修改過**的參數
- **重要**：只有這個區域的參數會被儲存到資料庫！

### 🎨 顏色與圖示說明

#### **參數狀態圖示**
- ✅ **已完成**：該分類的參數已全部設定完成
- ⚠️ **進行中**：該分類的參數正在設定中
- ❌ **未開始**：該分類的參數尚未開始設定
- 🔧 **需要外部工具**：該分類需要使用校準工具

#### **背景顏色意義**
- ⬜ **白色**：參考區中未被使用的參數
- 🟢 **淺綠色**：參考區中已複製但未修改的參數
- 🔵 **淺藍色**：參考區中已複製且已修改的參數

## 完整操作流程

### 📝 第一步：確認目標料號

#### 情況一：新料號設定
1. 在主視窗中，點擊「料號設定」選單
2. 進行料號新增
3. 輸入新料號資訊
4. 點擊「儲存」後，系統有了此料號的基本資訊，可開始設定此料號的參數

#### 情況二：舊料號參數調整
1. 直接從主視窗選擇要調整參數的料號
2. 開啟檢測參數設定視窗
3. 可選擇自己作為參考料號進行微調

### 📝 第二步：選擇參考料號

1. 在視窗上方找到「來源料號」下拉選單
   ```
   來源料號: [選擇舊料號 ▼]
   ```

2. 點擊下拉選單，會顯示所有已設定的料號清單：
   ```
   ▼ 選擇來源料號
   ├─ 10201120714TP  
   ├─ 10214090420T  
   ├─ 10215112360T   ← 假設選這個
   └─ ...
   ```

3. **如何選擇合適的來源料號？**
   - ✅ 選擇**產品外型最相似**的料號
   - ✅ 選擇**尺寸接近**的料號
   - ✅ 選擇**檢測需求相似**的料號
   - ✅ 選擇**PTFE同色**的料號
   - ❌ 避免選擇完全不同類型的產品

4. 選擇後，「參考區」會立即載入該料號的所有參數

---

### 📝 第三步：逐一設定五大分類參數

系統將參數分為五大類別，建議按照以下順序設定：

```
1. 相機參數 → 2. 位置參數 → 3. 檢測參數 → 4. 時間參數 → 5. 測試驗證
```

#### **為什麼要按照這個順序？**
- **相機參數**影響影像品質，必須先設定
- **位置參數**決定檢測區域，需要好的影像才能校準
- **檢測參數**各種檢測細節使用的參數
- **時間參數**影響推料時機的參數
- **測試驗證**是最後確認所有設定是否可用

#### **每個分類的操作流程（通用）**

##### Step A：檢視參考區參數
1. 點擊分類標籤（例如：「相機參數」）
2. 參考區會顯示來源料號的參數清單：
   ```
   參考區（來源料號：10215112360T）
   ┌────────────────────────────────┐
   │ ☐ exposure     = 5000                                     │ ⬜ 白色背景（未使用）
   │ ☐ gain         = 10                                            │
   │ ☐ delay        = 100                                          │
   └────────────────────────────────┘
   ```

##### Step B：選擇要複製的參數
有三種方式選擇參數：

**方式 1：手動勾選**
- 在參考區中，點擊參數前面的「☐」核取方塊
- 選中的參數會變成「☑」

**方式 2：使用快速選取按鈕**
```
[全選]    ← 勾選參考區中的所有參數
[全不選]  ← 取消所有勾選
[反選]    ← 勾選變不勾選，不勾選變勾選
```

**方式 3：智能選擇**
- 系統會自動判斷哪些參數**必須設定**
- 必須設定的參數會以**粗體**或**紅字**顯示

##### Step C：複製參數到「已新增未修改區」
1. 勾選好要複製的參數後
2. 點擊「**複製到新增區 →**」按鈕
3. 參數會出現在中間的「已新增未修改區」
4. 參考區中該參數的背景會變成**淺綠色** 🟢


##### Step D：修改參數值

在「已新增未修改區」中修改參數：

1. **直接在表格中編輯**
   - 點擊參數的「值」欄位
   - 輸入新的數值
   - 按 Enter 確認

2. **數值輸入注意事項**
   - 整數參數：直接輸入數字（例如：`5000`）
   - 小數參數：使用小數點（例如：`3.14`）
   - 布林參數：輸入 `true` 或 `false`
   - 字串參數：直接輸入文字

3. **修改範圍限制**
   - 如果輸入超出範圍的值，系統會顯示錯誤訊息
   - 參考「參數說明」欄位，了解合理範圍

##### Step E：標記為已修改
修改完參數值後：

1. 勾選已修改的參數
2. 點擊「**標記為已修改 →**」按鈕
3. 參數會移動到「已新增已修改區」
4. 參考區中該參數的背景會變成**淺藍色** 🔵


##### Step F：儲存此分類
1. 確認「已新增已修改區」的參數都正確
2. 點擊視窗下方的「**儲存此分類**」按鈕
3. 系統會將參數寫入資料庫
4. 分類標籤會顯示 ✅ 圖示

#### **復原操作**

如果設定錯誤，可以復原：

**復原修改（從已修改→未修改）**
1. 在「已新增已修改區」勾選要復原的參數
2. 點擊「**← 復原修改**」按鈕
3. 參數會移回「已新增未修改區」，恢復原始值

**完全移除參數（從新增區→刪除）**
1. 在「已新增未修改區」或「已新增已修改區」勾選要移除的參數
2. 點擊「**← 移除選取項目**」按鈕
3. 參數會從新增區消失
4. 參考區中該參數的背景恢復成**白色** ⬜

---

### 📝 第四步：儲存並完成設定

#### **儲存方式選擇**

**方式 1：分次儲存（推薦）**
- 每完成一個分類就點擊「**儲存此分類**」
- **優點**：不會因為關閉視窗而遺失已設定的參數
- **適用**：設定複雜、需要長時間調整的情況

**方式 2：一次儲存全部**
- 等所有分類都設定完成
- 點擊「**儲存全部並關閉**」按鈕
- **優點**：一次完成所有操作
- **適用**：參數簡單、快速設定的情況

#### **儲存確認**

儲存時，系統會顯示確認對話框：

```
┌─────────────────────────────────┐
│  ⚠️ 確認儲存                                                     │
├─────────────────────────────────┤
│  即將儲存以下參數到資料庫：                             │
│                                                                        │
│  目標料號：10215112360T                                   │
│  分類：相機參數                                                │
│  參數數量：3                                                     │
│                                                                        │
│  - exposure = 6000                                           │
│  - gain = 12                                                      │
│  - delay = 150                                                   │
│                                                                         │
│  確定要儲存嗎？                                                │
├─────────────────────────────────┤
│     [確定]        [取消]                                           │
└─────────────────────────────────┘
```

點擊「確定」後：
- ✅ 參數已成功儲存
- 📊 進度條更新
- 🎯 分類狀態圖示更新

---

### 📝 第五步：匯出工作階段（可選）

#### **什麼是工作階段？**
工作階段是一個 JSON 檔案，記錄了：
- 您選擇的來源料號
- 所有分類的參數設定
- 進度狀態

#### **為什麼要匯出？**
- 📄 **備份**：防止設定遺失
- 🔄 **快速復原**：下次可以直接載入
- 👥 **分享設定**：給其他工程師參考

#### **如何匯出？**
1. 點擊「**匯出工作階段**」按鈕
2. 選擇儲存位置
3. 檔案會自動命名為：`param_session_{料號}.json`
4. 預設位置：`bin\x64\Release\setting\`

#### **如何匯入？**
1. 開啟參數設定視窗
2. 點擊「**匯入工作階段**」按鈕
3. 選擇之前匯出的 JSON 檔案
4. 系統會自動載入所有設定

## 五大參數分類詳解

### 1.相機參數（Camera）

#### **參數說明**

| 參數名稱 | 說明 | 典型範圍 | 調整建議 |
|---------|------|---------|---------|
| `exposure` | 曝光時間（微秒） | 1000-10000 | 產品太暗→增加 產品太亮→減少 |
| `gain` | 增益值 | 0-20 | 影像雜訊多→減少 影像太暗→增加 |
| `delay` | 拍照延遲（毫秒） | 50-500 | 依據產品大小調整 |

#### **設定流程**

##### Step 1：複製參考參數
```
1. 點擊「相機參數」標籤
2. 在參考區勾選：exposure, gain, delay
3. 點擊「複製到新增區 →」
```

##### Step 2：調整曝光值（exposure）
```
1. 先使用來源料號的值測試拍照
2. 觀察影像效果：
   - 影像太暗 → exposure 增加 500-1000
   - 影像太亮 → exposure 減少 500-1000
   - 影像模糊 → exposure 減少（避免運動模糊）
3. 建議範圍：3000-8000 微秒
```

##### Step 3：調整增益值（gain）
```
1. 在曝光值調整完後再調整增益
2. 觀察影像雜訊：
   - 雜訊太多 → gain 減少 1-2
   - 亮度不足 → gain 增加 1-2
3. 建議範圍：5-15
```

##### Step 4：調整拍照延遲（delay）
```
1. 延遲時間 = PLC觸發訊號 → 相機實際拍照的等待時間
2. 考慮因素：
   - 產品移動速度
   - 輸送帶震動穩定時間
3. 建議範圍：100-300 毫秒
```

##### Step 5：測試拍照效果
```
1. 點擊「測試拍照」按鈕（如果有）
2. 或在主視窗中手動觸發拍照
3. 檢查影像品質：
   ✅ 清晰、亮度適中、無雜訊
   ❌ 模糊、過暗、過亮、雜訊多 → 回 Step 2 調整
```

##### Step 6：標記並儲存
```
1. 勾選已調整的參數
2. 點擊「標記為已修改 →」
3. 點擊「儲存此分類」
```

###  2.位置參數（Position）

#### **參數說明**

位置參數決定了：
- 產品在影像中的位置
- 檢測區域（ROI）的範圍
- 圓形、矩形、倒角等幾何特徵

| 參數類型 | 參數名稱範例 | 說明 |
|---------|-------------|------|
| 圓心座標 | `centerX1`, `centerY1` | 圓形檢測區域的中心點 |
| 半徑 | `radius1`, `radius2` | 圓形檢測區域的半徑 |
| 倒角位置 | `chamferX`, `chamferY` | 倒角特徵的位置 |
| ROI 位置 | `position1X`, `position1Y` | 矩形檢測區域的左上角座標 |

#### **設定流程（使用校準工具）**

##### Step 1：準備測試樣品
```
1. 放置一個良品樣品在檢測位置
2. 確認照明正常
3. 確認相機參數已設定完成（影像品質良好）
```

##### Step 2：開啟圓形校準工具
```
1. 點擊「位置參數」標籤
2. 點擊「🔧 開啟位置校正工具」按鈕
3. 選擇「圓形校準」（CircleCalibration）
```

##### Step 3：拍攝參考影像
```
1. 在校準工具視窗中
2. 點擊「拍攝影像」或「載入影像」按鈕
3. 影像會顯示在視窗中
```

### 3.檢測參數（Detection）

#### **參數說明**

檢測參數決定了系統如何判斷產品合格或不合格。

| 參數類型 | 參數名稱範例 | 說明 | 典型範圍 |
|---------|-------------|------|---------|
| 物體偏移 | `objBias1`, `objBias2` | 物體位置容許偏移量（像素） | 0-50 |
| 對比度 | `contrastMin`, `contrastMax` | 影像對比度範圍 | 0-255 |
| 白平衡 | `whiteRef1`, `whiteRef2` | 白色基準值 | 200-255 |
| 間隙閾值 | `gapThresh1` | 間隙寬度閾值（像素） | 1-20 |
| 輪廓參數 | `contourMinArea`, `contourMaxArea` | 輪廓面積範圍 | 10-10000 |

#### **設定流程**

##### Step 1：複製基本檢測參數
```
1. 點擊「檢測參數」標籤
2. 在參考區，建議先複製這些參數：
   ☑ threshold1
   ☑ threshold2
   ☑ objBias1
   ☑ objBias2
3. 點擊「複製到新增區 →」
```

##### Step 2：使用校準工具精細調整

**對比校準工具（ContrastCalibration）**
```
1. 點擊「對比檢測校正」
2. 載入測試影像（取過ROI的影像）
3. 調整對比度滑桿，觀察效果
4. 找到能清楚區分缺陷和正常區域的對比度值
5. 儲存參數
```

**像素校準工具（PixelCalibration）**
```
用途：找到檢測精度

1. 點擊「檢測經度校正」
2. 載入一張"原始圖像"
3. 對樣品畫出一條直徑，會得到檢測精度
4. 儲存參數
```

**白平衡校準工具（WhiteCalibration）**
```
用途：設定白色基準，用於顏色檢測

1. 點擊「白色像素校正」
2. 選擇多張 "原始圖片"
3. 開始分析
4. 若標準差在0.5以下，直接套用推薦值（儲存參數）
5. 若在0.5以上，請聯絡工程師
```

**間隙閾值校準工具（GapThreshCalibration）**
```
用途：設定間隙寬度檢測閾值

1. 點擊「開口閾值校正」
2. 載入有間隙缺陷的影像
3. 滑動滑桿後點擊分析（每次滑動都要重新點擊）
4. 找到滿意的結果後儲存相對應值
5. 儲存參數
```


##### Step 5：標記並儲存
```
1. 確認所有檢測參數都在「已新增已修改區」
2. 勾選所有參數
3. 點擊「標記為已修改 →」
4. 點擊「儲存此分類」
```

## 常見問題與解決方案

### ❓ Q1：參考區的參數消失了怎麼辦？

**A：參考區的參數永遠不會消失**

如果您看不到參考區的參數，可能是：

**原因1：沒有選擇來源料號**
```
解決方法：
1. 檢查視窗上方「來源料號」下拉選單
2. 確認有選擇一個料號
3. 如果是空白，重新選擇來源料號
```

**原因2：來源料號沒有參數**
```
解決方法：
1. 選擇另一個有參數的料號
2. 或先為來源料號設定參數
```

**原因3：視窗沒有正確載入**
```
解決方法：
1. 關閉參數設定視窗
2. 重新開啟
```

---

### ❓ Q2：為什麼我編輯了參數，但儲存後沒有變化？

**A：可能是參數還在「已新增未修改區」**

**問題原因**
```
只有「已新增已修改區」的參數會被儲存到資料庫！
```

**解決方法**
```
1. 確認參數在「已新增已修改區」
2. 如果參數在「已新增未修改區」：
   - 勾選該參數
   - 點擊「標記為已修改 →」
   - 再點擊「儲存此分類」
```

---

### ❓ Q3：位置校準後，參考區的位置參數變成空白了？

**A：這是舊版本的 bug，已在新版本修正**

**如果您使用的是新版本**
```
→ 參考區的參數不會消失
→ 校準結果會直接新增到「已新增已修改區」
```

**如果您使用的是舊版本**
```
解決方法：
1. 校準前先複製位置參數到「已新增未修改區」
2. 然後再開啟校準工具
3. 校準完成後，參考區的參數會保留
```

---

### ❓ Q4：objBias 參數找不到在哪裡？

**A：objBias 在「檢測參數」分類中**

```
正確位置：
[檢測參數] 標籤 → 參考區 → 找到 objBias1, objBias2 等參數

常見誤解：
❌ objBias 不在「位置參數」分類中
   （雖然它和位置有關，但它是用來調整檢測容許度的）
```

---

### ❓ Q5：測試驗證時準確率很低怎麼辦？

**A：需要系統性地分析問題**

#### **Step 1：分析錯誤類型**

```
查看測試報告，找出：
- 誤判數量（良品 → 不良）
- 漏檢數量（不良 → 良品）

誤判多 → 閾值太嚴格
漏檢多 → 閾值太寬鬆
```

#### **Step 2：檢視異常分數分佈**

```
良品的異常分數應該 < 0.3
不良品的異常分數應該 > 0.7

如果分數分佈重疊：
良品：0.2, 0.3, 0.4, 0.5  ← 分數太高
不良品：0.4, 0.5, 0.6, 0.7 ← 分數太低

→ 表示 AI 模型的區別能力不足
→ 建議重新訓練模型
```

#### **Step 3：調整參數**

```
情況A：少量誤判（1-2個）
→ 微調閾值（每次調整 0.05）

情況B：大量誤判（>5個）
→ 檢查影像品質
→ 重新校準位置參數
→ 考慮調整相機參數

情況C：少量漏檢（1-2個）
→ 降低閾值（每次降低 0.05）
→ 檢查不良品的缺陷是否明顯

情況D：大量漏檢（>5個）
→ AI 模型可能不適用於這種缺陷
→ 建議重新訓練模型，加入更多不良品樣本
```

---

### ❓ Q6：儲存時提示「資料庫鎖定」錯誤？

**A：資料庫正在被其他程式使用**

**原因**
```
- 主程式正在寫入資料庫
- 其他參數設定視窗同時開啟
- 資料庫檔案被防毒軟體掃描
```

**解決方法**
```
1. 稍等 5-10 秒後重試
2. 確認沒有開啟多個參數設定視窗
3. 如果持續發生：
   - 關閉主程式
   - 重新開啟參數設定視窗
```

---

### ❓ Q7：工作階段檔案載入後，參數沒有顯示？

**A：工作階段檔案可能損壞或格式錯誤**

**檢查方法**
```
1. 用記事本開啟工作階段檔案：
   bin\x64\Release\setting\param_session_{料號}.json

2. 檢查格式是否正確（應該是 JSON 格式）：
   {
     "targetType": "10215112360T",
     "sourceType": "10215112360T",
     "parameters": { ... }
   }

3. 如果格式錯誤：
   - 刪除該檔案
   - 重新設定參數並匯出
```

---

### ❓ Q8：參數值輸入後變成 0 或消失？

**A：可能輸入了不合法的值**

**常見原因**
```
1. 輸入了文字而不是數字
   ❌ exposure = "很亮"
   ✅ exposure = 6000

2. 輸入了超出範圍的值
   ❌ gain = 999
   ✅ gain = 12

3. 整數參數輸入了小數
   ❌ delay = 100.5
   ✅ delay = 100

4. 小數參數使用了逗號而不是句點
   ❌ threshold = 0,5
   ✅ threshold = 0.5
```

**解決方法**
```
1. 參考「參數說明」欄位的範圍
2. 使用正確的數值格式
3. 如果不確定，先複製參考區的值，再微調
```

---

### ❓ Q9：進度條顯示 100% 但仍有分類未完成？

**A：可能是進度計算的視覺誤差**

**檢查方法**
```
1. 查看狀態列的分類圖示：
   ✅ 相機(3/3)   ← 已完成
   ⚠️ 位置(5/8)   ← 進行中
   🔧 檢測(需校準) ← 需要外部工具
   ❌ 時間(0/4)   ← 未開始

2. 逐一檢查每個分類：
   - 點擊分類標籤
   - 查看「已新增已修改區」是否有參數
   - 確認已點擊「儲存此分類」

3. 如果某個分類確實完成但圖示錯誤：
   - 關閉視窗重新開啟
   - 或忽略圖示，直接測試驗證
```

## 進階技巧

### 💡 技巧 1：快速複製相似料號

如果您有多個非常相似的料號要設定：

```

2. 匯出工作階段：
   檔名：param_session_料號A.json

   param_session_料號B.json

4. 用記事本編輯 料號B.json：
   - 將 "targetType" 改為 料號B
   - 微調不同的參數值

5. 開啟料號B的參數設定視窗
6. 匯入 料號B.json
7. 只調整差異的參數
8. 儲存
```

contourMaxArea          - 輪廓最大面積
```
