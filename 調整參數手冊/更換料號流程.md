## 更換料號概念

### 什麼是更換料號？

更換料號是指從當前檢測的料號切換到另一個**已存在**的料號。與「新增料號」不同，更換料號的參數已經設定完成，主要工作是：
1. **調整硬體環境**：確保新料號能正常通過軌道和定位
2. **驗證影像品質**：確認拍攝位置和ROI準確

### 更換料號 vs 新增料號

| 項目 | 更換料號 | 新增料號 |
|------|----------|----------|
| **參數設定** | 已完成（從資料庫載入） | 需要從零設定或複製來源料號 |
| **硬體調整** | 需要（軌道、焦距） | 通常不需要 |
| **校正工作** | 僅需微調驗證 | 需要完整校正流程 |
| **所需時間** | 30-45分鐘 | 60-90分鐘 |
| **主要風險** | 硬體未調整好、對位不準 | 參數設定錯誤 |

### 兩大核心任務

**任務1：硬體環境正確**
- 調整直震、圓震軌道，確保產品能順利通過
- 調整相機焦距，確保影像清晰
- 調整轉盤位置，確保產品對準拍攝區域

**任務2：影像正確**
- 驗證拍攝位置準確（產品在畫面中央）
- 驗證ROI對齊正確
- 驗證圓心偵測輔助線貼合產品

## 前置準備

### 確認事項

在開始更換料號前，請確認以下事項：

**1. 料號已建立**
- [ ] 目標料號已在系統中建立
- [ ] 目標料號的參數已設定完成
- [ ] 可在料號下拉選單中找到目標料號

**2. 樣品準備**
- [ ] 已準備足夠的測試樣品（建議15-20個良品）
- [ ] 樣品清潔，無灰塵或異物
- [ ] 樣品尺寸、形狀符合目標料號規格

**3. 系統狀態**
- [ ] 系統運行正常，無異常錯誤訊息
- [ ] 相機正常運作
- [ ] PLC通訊正常

**4. 工具準備**
- [ ] 調整軌道的工具（如扳手、螺絲起子）
- [ ] 清潔相機鏡頭的工具（如鏡頭紙）
- [ ] 測試樣品的容器

### 預期調整項目

根據新料號與舊料號的差異，可能需要調整：

| 差異項目 | 需調整內容 | 難度 |
|----------|------------|------|
| **產品尺寸不同** | 直震軌道寬度、圓震軌道間隙 | 中 |
| **產品高度不同** | 相機焦距 | 高 |
| **產品外徑不同** | 轉盤速度、圓心偵測參數 | 低 |
| **產品材質不同** | 通常不需調整硬體 | 無 |

## 詳細操作步驟

### 步驟1：調整硬體軌道

#### 目的

確保新料號的產品能夠順利通過直震和圓震軌道，到達檢測位置。

#### 什麼是直震、圓震？

- **直震**：直線振動送料器，將產品從料斗輸送到圓震
- **圓震**：圓形振動送料器，將產品排列整齊並送入轉盤

#### 操作步驟

**1.1 調整直震軌道**

調整目標：產品能穩定通過，不會卡住或掉落

1. 停止系統運行
2. 將測試樣品放入直震料斗
3. 觀察產品在軌道上的移動狀況
4. 調整軌道寬度：
   - 產品卡住 → 放寬軌道（鬆開固定螺絲，向外調整）
   - 產品不穩定或掉落 → 縮窄軌道
   - 理想狀態：產品「剛好」通過，略有接觸但不會卡住

**1.2 調整圓震軌道**

調整目標：產品能順利進入圓震，並整齊排列

1. 檢查圓震軌道間隙
2. 測試產品是否能順利進入並旋轉
3. 調整間隙：
   - 產品卡在入口 → 增加間隙
   - 產品在圓震內堆疊或翻倒 → 調整引導片角度
   - 理想狀態：產品單個進入，排列整齊

**1.3 測試送料流程**

1. 手動啟動直震和圓震
2. 放入10個測試樣品
3. 觀察送料過程是否順暢
4. 確認產品能穩定到達轉盤位置

#### 檢查點
- [ ] 產品能順利通過直震軌道
- [ ] 產品能順利通過圓震軌道
- [ ] 送料過程穩定，無卡料或掉料
- [ ] 產品到達轉盤位置時姿態正常

#### 注意事項
- 調整時務必停止系統運行，避免危險
- 調整幅度從小到大，避免過度調整
- 記錄調整前的位置，以便必要時復原

---

### 步驟2：設定檢測料號

#### 操作路徑
```
主介面 → 料號下拉選單 → 選擇目標料號
```

#### 操作步驟

1. 在主介面找到「**料號選擇**」下拉選單
2. 點擊下拉選單，顯示所有已建立的料號
3. 選擇要更換的目標料號（例如：`10214090420T`）
4. 確認主介面顯示的料號已更新為目標料號

#### 檢查點
- [ ] 已選擇正確的目標料號
- [ ] 主介面顯示的料號名稱正確
- [ ] 系統已載入該料號的參數設定

#### 注意事項
- 確認選擇的料號正確，避免使用錯誤的參數進行檢測
- 選擇料號後，系統會自動從資料庫載入該料號的所有參數

### 步驟3：進入調機模式並對準樣品

#### 目的

在調機模式下手動控制轉盤，將產品移動到相機正下方，並進行對位。

#### 什麼是調機模式？

調機模式是手動操作環境，允許操作員：
- 手動控制轉盤旋轉
- 即時查看相機畫面
- 調整產品位置
- 不執行自動檢測

#### 操作路徑
```
主介面 → 模式切換 → 調機模式
```

#### 操作步驟

**3.1 進入調機模式**

1. 在主介面找到「**模式切換**」按鈕或下拉選單
2. 選擇「**調機模式**」（可能標示為「Manual Mode」）
3. 確認系統狀態顯示為「調機模式」

**3.2 手動放置樣品**

1. 將一個測試樣品放在轉盤上
2. 或透過直震、圓震送料，讓樣品進入轉盤

**3.3 手動旋轉轉盤**

1. 找到「**手動轉盤**」控制按鈕（可能是旋轉箭頭圖示）
2. 點擊按鈕，使轉盤旋轉
3. 將樣品移動到相機正下方
4. 觀察主介面的影像顯示區域

**3.4 查看輔助線**

1. 當樣品移動到相機視野內，螢幕上會顯示產品影像
2. 畫面上會有**圓形輔助線**（或其他形狀的參考線）
3. 輔助線的作用：
   - 顯示理想的產品位置
   - 幫助操作員判斷產品是否對準
   - 基於該料號的圓心偵測參數繪製

**3.5 初步對準**

1. 觀察產品與輔助線的相對位置
2. 如果產品偏離輔助線：
   - 繼續手動旋轉轉盤
   - 調整產品到接近輔助線的位置
3. 這個階段不需要完美對齊，大致接近即可

#### 檢查點
- [ ] 已成功進入調機模式
- [ ] 樣品已放置在轉盤上
- [ ] 螢幕顯示產品影像
- [ ] 可以看到圓形輔助線
- [ ] 產品大致在畫面中央

#### 注意事項
- 調機模式下，系統不會自動檢測，可以放心調整
- 輔助線是根據該料號的位置參數顯示，不同料號可能不同

### 步驟4：調整相機焦距

#### 目的

調整相機鏡頭焦距，使產品影像清晰銳利。

#### 為什麼需要調整焦距？

不同料號的產品高度可能不同，導致產品與相機鏡頭的距離改變，需要重新對焦才能獲得清晰影像。

#### 操作步驟

**4.1 檢查當前影像清晰度**

1. 在調機模式下，觀察螢幕上的產品影像
2. 檢查影像是否清晰：
   - ✓ 清晰：邊緣銳利，細節清楚
   - ✗ 模糊：邊緣朦朧，細節不清

**4.2 調整焦距（硬體操作）**

⚠️ **警告：調整焦距需要實際操作相機鏡頭，請小心操作**

1. 找到相機鏡頭上的**對焦環**（通常在鏡頭中段）
2. 一邊觀察螢幕影像，一邊旋轉對焦環：
   - 順時針旋轉
   - 逆時針旋轉
3. 找到影像最清晰的位置
4. 鎖定對焦環（如果有鎖定機構）

**4.3 反覆微調**

1. 手動旋轉轉盤，讓產品離開視野
2. 再次旋轉轉盤，讓產品回到視野
3. 檢查影像是否仍然清晰
4. 如果不清晰，重複步驟4.2

**4.4 多點驗證**

1. 使用多個樣品（3-5個）測試
2. 確保所有樣品的影像都清晰
3. 如果某些樣品清晰、某些模糊，可能是：
   - 樣品高度不一致（品質問題）
   - 轉盤高度不平（機械問題）

#### 影像清晰度判定標準

| 項目 | 清晰（合格） | 模糊（不合格） |
|------|-------------|---------------|
| **邊緣** | 銳利、分明 | 朦朧、虛化 |
| **細節** | 可見細微特徵（如紋理、標記） | 細節模糊不清 |
| **對比度** | 高對比，黑白分明 | 灰濛濛，對比度低 |

#### 檢查點
- [ ] 產品影像清晰，邊緣銳利
- [ ] 多個樣品測試結果一致
- [ ] 對焦環已鎖定（如果有鎖定機構）

#### 注意事項
- 調整焦距時動作要輕柔，避免損壞鏡頭
- 如果無法調整到清晰狀態，可能是鏡頭髒污，需清潔鏡頭
- 不同站點的相機可能需要分別調整

---

### 步驟5：精確對準輔助線

#### 目的

透過反覆調整轉盤和觀察影像，使產品精確對準輔助線，確保後續檢測時產品位置穩定。

#### 操作步驟

**5.1 觀察當前對齊狀況**

1. 在調機模式下，觀察產品與輔助線的位置關係
2. 判斷偏差方向和程度：
   - 產品中心是否與輔助線中心重合？
   - 產品邊緣是否與輔助線貼合？
   - 偏差方向（左、右、上、下）

**5.2 手動調整轉盤**

1. 如果產品未對準，手動旋轉轉盤
2. 微調產品位置
3. 重新觀察對齊情況

**5.3 調整焦距（如需要）**

1. 如果在對齊過程中發現影像不清晰
2. 重複步驟4的焦距調整
3. 確保影像清晰後再繼續對齊

**5.4 反覆調整**

調整流程（循環直到滿意）：
1. 旋轉轉盤 → 產品離開視野
2. 再次旋轉 → 產品回到視野
3. 檢查對齊情況
4. 如未對準 → 繼續調整
5. 如已對準 → 測試穩定性

**5.5 測試穩定性**

1. 讓產品離開視野
2. 使用不同的樣品（3-5個）
3. 每個樣品都旋轉到視野內
4. 檢查每個樣品是否都能對準輔助線
5. 如果有樣品無法對準，可能需要調整參數（見步驟9）

#### 對齊標準

**理想對齊：**
- 產品中心與輔助線中心重合
- 產品輪廓與輔助線貼合（誤差<5像素）
- 多個樣品測試結果一致

**可接受：**
- 產品中心與輔助線中心偏差<10像素
- 產品輪廓與輔助線基本貼合
- 偏差方向一致（可透過參數微調）

**不可接受：**
- 偏差>20像素
- 每個樣品的偏差方向不一致
- 產品部分超出輔助線範圍

#### 檢查點
- [ ] 產品中心與輔助線中心對齊
- [ ] 產品輪廓與輔助線貼合
- [ ] 多個樣品測試結果一致
- [ ] 偏差在可接受範圍內（<10像素）

#### 注意事項
- 這個階段的對齊是初步的，後續會使用圓心校正工具進行精確微調
- 如果無法對齊，可能是：
  - 轉盤速度不當
  - 產品尺寸與參數設定不符
  - 需要調整位置參數

### 步驟6：切換至檢測模式

#### 目的

退出調機模式，進入正式檢測模式，並執行異常復歸，準備進行測試檢測。

#### 操作路徑
```
關閉介面 → 停止調機 → 模式切換 → 檢測模式 → 異常復歸
```

#### 操作步驟

**6.1 關閉調機介面**

1. 如果有開啟任何調機相關的視窗或工具，先關閉它們
2. 回到主介面

**6.2 停止調機**

1. 找到「**停止調機**」按鈕
2. 點擊按鈕，停止調機模式
3. 系統會停止手動控制，準備切換模式

**6.3 切換至檢測模式**

1. 找到「**模式切換**」按鈕或下拉選單
2. 選擇「**檢測模式**」或「**自動模式**」
3. 確認系統狀態顯示為「檢測模式」

**6.4 執行異常復歸**

什麼是異常復歸？
- 系統初始化程序
- 復位所有計數器和狀態
- 清除暫存的錯誤訊息
- 與PLC進行通訊確認

如何執行：
1. 找到「**異常復歸**」按鈕（可能在主介面或設定選單中）
2. 點擊按鈕
3. 等待系統完成復歸（通常3-10秒）
4. 確認系統狀態顯示為「就緒」或「待機」

#### 檢查點
- [ ] 已成功退出調機模式
- [ ] 已切換至檢測模式
- [ ] 異常復歸執行成功
- [ ] 系統狀態顯示「就緒」或「待機」
- [ ] 無任何錯誤訊息

#### 注意事項
- 異常復歸會清除計數器，如果需要保留計數，請先記錄
- 如果異常復歸失敗，檢查PLC連線是否正常

---

### 步驟7：啟用影像儲存並測試檢測

#### 目的

儲存檢測過程中的ROI影像，用於後續的圓心校正和參數驗證。

#### 操作路徑
```
主介面 → 資料儲存 → 勾選 ROI → 放料測試 → 開始檢測
```

#### 操作步驟

**7.1 開啟資料儲存設定**

1. 在主介面找到「**資料儲存**」按鈕或選單
2. 點擊開啟資料儲存設定視窗

**7.2 啟用ROI儲存**

1. 在資料儲存設定中找到「**ROI**」選項
2. 勾選「**ROI**」
3. 建議同時勾選「**Origin**」（原始影像），方便後續分析
4. 確認設定已儲存

**7.3 準備測試樣品**

1. 準備10-20個良品樣品
2. 將樣品放入直震料斗

**7.4 設定介面參數**

根據需要設定以下參數（如果之前已設定，可跳過）：
- 檢測速度
- 儲存條件（全部儲存或僅儲存NG品）
- 其他相關參數

**7.5 開始檢測**

1. 點擊「**開始檢測**」按鈕
2. 系統會自動：
   - 啟動直震、圓震送料
   - 旋轉轉盤
   - 拍攝影像
   - 執行檢測
   - 儲存影像
3. 觀察主介面的即時影像顯示
4. 讓10-20個樣品完成檢測

**7.6 停止檢測**

1. 待測試樣品檢測完成後
2. 點擊「**停止檢測**」按鈕

#### 檢查點
- [ ] ROI儲存選項已勾選
- [ ] 已放入10-20個測試樣品
- [ ] 檢測過程順利，無卡料或異常
- [ ] 影像已自動儲存

#### 注意事項
- 這個階段的檢測結果不一定準確，主要目的是收集影像
- 如果檢測過程中出現送料問題，回到步驟1調整軌道
- 確保儲存足夠的影像供後續校正使用

---

### 步驟8：查看拍攝影像

#### 目的

檢查剛才檢測過程中拍攝的影像，確認影像品質和位置。

#### 操作路徑
```
主介面 → 檔案 → 開啟圖檔 → 選擇日期和料號
```

#### 操作步驟

**8.1 開啟圖檔瀏覽**

1. 點擊主介面左上角的「**檔案**」選單
2. 在下拉選單中選擇「**開啟圖檔**」或「**影像瀏覽**」
3. 系統會開啟檔案瀏覽視窗

**8.2 定位到當天資料**

1. 在檔案瀏覽視窗中，找到日期資料夾
2. 選擇當日日期（例如：`2025-11-27`）
3. 進入日期資料夾

**8.3 選擇料號資料夾**

1. 在日期資料夾中，找到剛才設定的料號
2. 點擊進入料號資料夾
3. 會看到以下子資料夾：
   - `origin\` - 原始影像
   - `ROI_1\` - 站點1的ROI影像
   - `ROI_2\` - 站點2的ROI影像
   - `ROI_3\` - 站點3的ROI影像
   - `ROI_4\` - 站點4的ROI影像

**8.4 瀏覽ROI影像**

1. 進入各個ROI資料夾
2. 查看剛才拍攝的影像
3. 確認影像時間戳記符合剛才的檢測時間

**8.5 檢查影像品質**

逐一檢查影像，重點項目：

| 檢查項目 | 良好 | 需改善 |
|----------|------|--------|
| **清晰度** | 邊緣銳利，細節清楚 | 模糊、殘影 → 調整曝光或焦距 |
| **產品位置** | 產品在畫面中央 | 偏左或偏右 → 需參數調整 |
| **ROI對齊** | ROI框準確包含產品 | ROI偏移 → 需參數調整 |
| **亮度** | 亮度適中，無過曝 | 過亮或過暗 → 調整相機參數 |

**8.6 記錄問題**

1. 如果發現影像有問題，記錄：
   - 哪個站點有問題
   - 問題類型（位置偏移、模糊、亮度等）
   - 偏移方向和大致程度
2. 這些資訊在下一步驟（圓心校正）會用到

#### 影像儲存路徑結構

```
bin\x64\Release\report\
└── {日期}\
    └── {料號}\
        ├── origin\          (原始影像)
        ├── ROI_1\           (站點1的ROI)
        ├── ROI_2\           (站點2的ROI)
        ├── ROI_3\           (站點3的ROI)
        └── ROI_4\           (站點4的ROI)
```

- [ ] 已找到當日日期的資料夾
- [ ] 已找到目標料號的資料夾
- [ ] 已查看各站點的ROI影像
- [ ] 影像品質符合標準或已記錄問題

#### 注意事項
- 如果找不到影像，確認「資料儲存」的ROI選項是否已勾選
- 如果影像數量不符，可能是檢測過程中有異常
- 影像檔名通常包含時間戳記和站點ID，方便識別

### 步驟9：使用圓心校正工具進行微調

#### 目的

使用圓心校正工具，根據實際拍攝的影像進行精確參數調整，確保輔助線與產品完美貼合。

#### 什麼是圓心校正工具？

圓心校正工具是用於調整產品圓心偵測參數的專用工具，可以：
- 載入實際拍攝的影像
- 顯示當前的圓心偵測結果和輔助線
- 調整偵測參數並即時預覽效果
- 儲存調整後的參數到資料庫

#### 操作路徑
```
主介面 → 設定 → 圓心校正工具
```

#### 操作步驟

**9.1 開啟圓心校正工具**

1. 點擊主介面的「**設定**」選單
2. 選擇「**圓心校正工具**」
3. 圓心校正工具視窗開啟

**9.2 載入測試影像**

1. 在圓心校正工具中，點擊「**載入影像**」或「**開啟檔案**」
2. 瀏覽到步驟8查看的影像資料夾
3. 選擇 `origin\` 資料夾中的影像（使用原始影像，不是ROI影像）
4. 選擇多張影像（建議5-10張）
5. 載入影像

**9.3 觀察當前偵測結果**

1. 工具會顯示載入的影像
2. 在影像上會顯示：
   - 偵測到的圓心位置（通常是紅色十字或點）
   - 輔助線（圓形或其他形狀）
   - ROI框的位置
3. 觀察輔助線與產品的貼合程度

**9.4 判斷需要調整的參數**

根據觀察結果，判斷調整方向：

| 觀察結果 | 需調整參數 | 調整方向 |
|----------|------------|----------|
| 輔助線整體偏左 | 圓心X座標偏移 | 向右調整（增加X值） |
| 輔助線整體偏右 | 圓心X座標偏移 | 向左調整（減少X值） |
| 輔助線整體偏上 | 圓心Y座標偏移 | 向下調整（增加Y值） |
| 輔助線整體偏下 | 圓心Y座標偏移 | 向上調整（減少Y值） |
| 輔助線大小不符 | 圓心半徑參數 | 調整半徑值 |
| 偵測不穩定 | 圓心偵測範圍 | 調整搜尋範圍 |

**9.5 調整參數**

1. 在圓心校正工具中找到參數調整區域
2. 可能包含的參數：
   - **圓心X偏移**：調整水平位置
   - **圓心Y偏移**：調整垂直位置
   - **圓心半徑**：調整輔助線大小
   - **搜尋範圍**：調整偵測區域
   - **偵測閾值**：調整偵測靈敏度
3. 調整參數，觀察即時預覽效果
4. 目標：輔助線與產品輪廓完美貼合

**9.6 精細調整**

⚠️ **重要：此步驟需要非常仔細**

1. 逐一檢查每張載入的影像
2. 確保所有影像的輔助線都能貼合產品
3. 如果某些影像貼合、某些不貼合：
   - 可能是產品位置變化較大
   - 需要找到平衡點，使所有影像的偏差最小
4. 反覆微調參數，直到滿意為止

**9.7 儲存調整結果**

1. 確認所有影像的輔助線都已貼合
2. 點擊「**儲存**」或「**套用**」按鈕
3. 調整後的參數會寫入資料庫
4. 關閉圓心校正工具

#### 調整精度要求

- **偏差容許範圍**：±5像素以內
- **理想狀態**：輔助線與產品邊緣完全重合
- **可接受狀態**：輔助線與產品邊緣偏差<5像素，且偏差一致

#### 參考文件

**詳細調整方法請參考：** `11.位置參數.docx`

該文件包含：
- 圓心校正工具的完整使用說明
- 各參數的詳細解釋
- 常見問題的調整範例
- 進階技巧

#### 檢查點
- [ ] 已開啟圓心校正工具
- [ ] 已載入5-10張原始影像（origin資料夾）
- [ ] 已調整參數使輔助線貼合產品
- [ ] 所有影像的偏差<5像素
- [ ] 已儲存調整結果

#### 注意事項
- **務必使用原始影像（origin），不是ROI影像**
- 調整時要查看多張影像，確保調整對所有影像都有效
- 調整幅度從小到大，避免過度調整
- 如果無法調整到理想狀態，可能需要重新檢查硬體對位（回到步驟1-5）

### 步驟10：再次測試驗證

#### 目的

在參數調整完成後，進行再次測試，驗證調整效果是否理想。

#### 操作步驟

**10.1 準備測試樣品**

1. 準備新的測試樣品（10-15個良品）
2. 不要使用步驟7已測試過的樣品（避免混淆）

**10.2 執行檢測**

1. 確認「資料儲存」的「ROI」仍然勾選
2. 將測試樣品放入直震料斗
3. 點擊「**開始檢測**」
4. 讓樣品完成檢測
5. 點擊「**停止檢測**」

**10.3 查看新拍攝的影像**

1. 重複步驟8的操作，查看新拍攝的影像
2. 重點檢查：
   - ROI對齊是否改善
   - 產品位置是否穩定
   - 所有影像的品質是否一致

**10.4 判斷調整效果**

檢查以下指標：

| 指標 | 理想狀態 | 可接受狀態 | 需重新調整 |
|------|----------|------------|------------|
| **位置偏差** | <5像素 | 5-10像素 | >10像素 |
| **ROI對齊** | 完全包含產品 | 產品在ROI內，略有邊距 | 產品被裁切或ROI偏移 |
| **穩定性** | 所有影像偏差<5像素 | 偏差<10像素 | 偏差不一致 |
| **影像清晰度** | 清晰銳利 | 稍微軟化但可接受 | 模糊 |

**10.5 處理結果**

**結果A：理想，繼續下一步驟**
- 位置偏差<5像素
- ROI對齊完美
- 所有影像品質一致

→ 進入步驟11

**結果B：可接受，但有改善空間**
- 位置偏差5-10像素
- ROI基本對齊，但不完美
- 大部分影像品質良好

→ 可選擇：
- 接受當前狀態，進入步驟11
- 或回到步驟9重新微調

**結果C：不理想，需重新調整**
- 位置偏差>10像素
- ROI對齊不佳
- 影像品質不穩定

→ 必須重新調整：
- 如果是位置偏差：回到步驟9重新校正參數
- 如果是硬體問題：回到步驟1-5檢查硬體對位
- 如果是影像模糊：回到步驟4調整焦距

#### 檢查點
- [ ] 已執行再次測試
- [ ] 已查看新拍攝的影像
- [ ] 位置偏差在可接受範圍內
- [ ] ROI對齊良好
- [ ] 影像品質穩定

#### 注意事項
- 這個步驟是驗證步驟，如果結果不理想，不要勉強進行下一步
- 良好的對位是檢測準確度的基礎，值得花時間調整到最佳狀態

### 步驟11：完成設定並開始正式檢測

#### 目的

完成所有設定，關閉影像儲存（避免占用磁碟空間），執行最後的異常復歸，準備進入正式生產檢測。

#### 操作步驟

**11.1 關閉影像儲存**

1. 點擊「**資料儲存**」按鈕或選單
2. 取消勾選「**ROI**」
3. 取消勾選「**Origin**」（如果之前勾選了）
4. 確認設定已儲存

為什麼要關閉影像儲存？
- 正式生產時不需要儲存所有影像
- 避免磁碟空間快速耗盡
- 提高系統效能

保留儲存的情況：
- 新料號初期生產，需要持續觀察
- 品質監控需要
- 客戶要求保留影像記錄

**11.2 確認介面設定**

檢查並確認以下設定：

| 設定項目 | 檢查內容 | 標準 |
|----------|----------|------|
| **料號** | 當前選擇的料號 | 與目標料號一致 |
| **檢測模式** | 系統模式 | 檢測模式（非調機模式） |
| **檢測速度** | 轉盤速度、送料速度 | 符合生產需求 |
| **NG品處理** | 不良品排出設定 | 已正確設定 |
| **計數器** | 生產計數器 | 已清零或設定正確 |
| **報警設定** | 異常報警條件 | 已設定 |

**11.3 執行最終異常復歸**

1. 點擊「**異常復歸**」按鈕
2. 等待系統完成復歸程序
3. 確認系統狀態顯示為「就緒」

為什麼要再次異常復歸？
- 清除測試過程中的暫存資料
- 重置計數器
- 確保系統狀態完全正常
- 與PLC重新同步

**11.4 開始正式檢測**

1. 準備生產用的樣品（確認數量充足）
2. 將樣品放入直震料斗
3. 點擊「**開始檢測**」
4. 系統進入正式生產模式

**11.5 初期監控**

在開始正式檢測後的前30-60分鐘，建議進行密切監控：

**監控項目：**
- 送料是否順暢（無卡料）
- 檢測結果是否合理（誤判率、檢出率）
- 系統是否穩定運行（無異常停機）
- NG品排出是否正確
- 計數是否正常累積

**處理異常：**
- 如果發現送料問題：停機調整軌道（回到步驟1）
- 如果發現檢測異常：檢查參數設定和影像品質
- 如果系統異常：查看日誌檔案，必要時聯繫技術支援

**11.6 記錄更換作業**

建議建立更換料號記錄，包含：
- 日期時間
- 操作員姓名
- 舊料號 → 新料號
- 調整項目（軌道、焦距、參數等）
- 測試結果（位置偏差、ROI對齊情況）
- 異常狀況（如果有）
- 完成時間

#### 檢查點
- [ ] 影像儲存已關閉（ROI、Origin都未勾選）
- [ ] 所有介面設定已確認正確
- [ ] 已執行最終異常復歸
- [ ] 系統狀態顯示「就緒」
- [ ] 已開始正式檢測
- [ ] 初期監控無異常
- [ ] 已記錄更換作業

## 驗證與測試

### 驗收標準

完成更換料號後，建議執行以下驗收測試，確保系統正常運作。

#### 基礎驗收（必須通過）

| 項目 | 標準 | 檢查方法 |
|------|------|----------|
| **送料順暢** | 無卡料、掉料 | 觀察30個樣品的送料過程 |
| **影像清晰** | 邊緣銳利，細節清楚 | 抽查5-10張影像 |
| **位置穩定** | 偏差<10像素 | 統計分析30張影像 |
| **ROI對齊** | 產品完整在ROI內 | 抽查所有站點的ROI影像 |
| **系統穩定** | 無當機、無異常錯誤 | 連續運行30分鐘 |

#### 進階驗收（建議通過）

| 項目 | 標準 | 檢查方法 |
|------|------|----------|
| **位置精度** | 偏差<5像素 | 統計分析50張影像 |
| **檢測準確度** | 誤判率<3% | 測試50個良品+10個瑕疵品 |
| **處理速度** | 符合生產節拍 | 計時測量每件處理時間 |
| **NG品排出** | 100%正確排出 | 測試10個瑕疵品 |

### 測試方法

#### 測試1：送料穩定性測試

1. 準備50個測試樣品
2. 連續放入直震料斗
3. 觀察送料過程
4. 記錄卡料或異常次數
5. 合格標準：0次異常

#### 測試2：位置穩定性測試

1. 啟用影像儲存（ROI和Origin）
2. 檢測30個樣品
3. 查看所有影像
4. 測量每張影像的產品中心位置
5. 計算標準差
6. 合格標準：標準差<10像素（理想<5像素）

#### 測試3：檢測準確度測試

1. 準備測試樣品：
   - 良品：50個
   - 已知瑕疵品：10個（涵蓋各類瑕疵）
2. 執行檢測並記錄結果
3. 計算指標：
   - 檢出率 = (檢出瑕疵品數 / 總瑕疵品數) × 100%
   - 誤判率 = (誤判良品數 / 總良品數) × 100%
4. 合格標準：檢出率≥95%，誤判率≤3%

## 常見問題排除

### 硬體相關問題

#### Q1: 產品在直震軌道卡住

**可能原因：**
- 軌道過窄
- 產品尺寸超出規格
- 軌道有異物或髒污

**解決方式：**
1. 停止系統
2. 取出卡住的產品
3. 檢查產品尺寸是否正常
4. 清潔軌道，移除異物
5. 適當放寬軌道寬度
6. 重新測試

---

#### Q2: 產品在圓震內堆疊或翻倒

**可能原因：**
- 圓震速度過快
- 引導片角度不當
- 產品形狀不規則

**解決方式：**
1. 調整圓震振動頻率（降低速度）
2. 調整引導片角度
3. 檢查產品品質（形狀是否規則）
4. 減少一次投料量

---

#### Q3: 影像持續模糊，無法調整清晰

**可能原因：**
- 相機鏡頭髒污
- 焦距調整範圍不足（產品高度差異過大）
- 相機故障

**解決方式：**
1. 清潔相機鏡頭（使用專用鏡頭紙）
2. 檢查產品高度是否在規格內
3. 檢查鏡頭是否損壞
4. 如果問題持續，聯繫技術支援
5. 考慮更換鏡頭或調整相機安裝高度

---

### 軟體相關問題

#### Q4: 無法找到目標料號

**可能原因：**
- 料號尚未建立
- 料號名稱輸入錯誤
- 資料庫連線問題

**解決方式：**
1. 確認料號是否已在系統中建立
2. 檢查料號名稱拼寫
3. 重新載入料號列表（可能需要重啟系統）
4. 檢查資料庫檔案是否存在（`mydb.sqlite`）

---

#### Q5: 異常復歸失敗

**可能原因：**
- PLC通訊異常
- 系統狀態錯誤
- 硬體未就緒

**解決方式：**
1. 檢查PLC連線（查看通訊指示燈）
2. 檢查序列埠設定
3. 重新啟動系統
4. 檢查PLC是否正常運作
5. 查看日誌檔案尋找錯誤訊息

---

#### Q6: 圓心校正工具無法載入影像

**可能原因：**
- 影像路徑錯誤
- 影像格式不支援
- 影像檔案損壞

**解決方式：**
1. 確認影像路徑正確
2. 確認使用origin資料夾的影像（不是ROI）
3. 檢查影像格式（應為JPG或PNG）
4. 嘗試使用其他影像
5. 重新拍攝影像

---

#### Q7: 調整參數後仍無法對齊

**可能原因：**
- 參數調整方向錯誤
- 硬體對位問題（步驟1-5未做好）
- 產品尺寸與資料庫設定不符

**解決方式：**
1. 確認參數調整方向是否正確
2. 回到步驟3-5，重新檢查硬體對位
3. 確認產品尺寸規格
4. 檢查是否選擇了正確的料號
5. 考慮重新建立料號參數

---

### 影像品質問題

#### Q8: 影像亮度不均

**可能原因：**
- 光源問題
- 相機參數不當
- 環境光干擾

**解決方式：**
1. 檢查光源是否正常（LED是否損壞）
2. 調整曝光時間和增益
3. 遮蔽環境光源
4. 清潔光源表面

---

#### Q9: ROI影像中產品被裁切

**可能原因：**
- 位置偏差過大
- ROI範圍設定過小
- 產品尺寸超出預期

**解決方式：**
1. 使用圓心校正工具調整位置參數
2. 檢查產品尺寸是否在規格內
3. 如果偏差持續，考慮調整ROI範圍（在參數設定中）
4. 回到步驟3-5重新對位

---

#### Q10: 不同站點影像品質差異大

**可能原因：**
- 各站點相機參數不同
- 各站點硬體狀況不同（光源、鏡頭）
- 轉盤高度不平

**解決方式：**
1. 分別檢查每個站點的相機和光源
2. 統一各站點的相機參數（如果可能）
3. 分別調整各站點的焦距
4. 檢查轉盤是否水平
5. 考慮硬體維護或更換

## 操作檢查清單

### 前置準備檢查

- [ ] 目標料號已在系統中建立
- [ ] 目標料號的參數已設定完成
- [ ] 已準備足夠的測試樣品（15-20個良品）
- [ ] 系統運行正常，無異常錯誤
- [ ] 已準備調整工具（扳手、螺絲起子等）

### 步驟1：硬體調整

- [ ] 已調整直震軌道寬度
- [ ] 已調整圓震軌道間隙
- [ ] 產品能順利通過軌道，無卡料
- [ ] 產品能穩定到達轉盤位置

### 步驟2：設定料號

- [ ] 已選擇正確的目標料號
- [ ] 主介面顯示的料號名稱正確
- [ ] 系統已載入該料號參數

### 步驟3：調機模式對準

- [ ] 已進入調機模式
- [ ] 樣品已放置在轉盤上
- [ ] 可以看到產品影像和輔助線
- [ ] 產品大致對準輔助線

### 步驟4：調整焦距

- [ ] 影像清晰，邊緣銳利
- [ ] 多個樣品測試結果一致
- [ ] 焦距已鎖定（如有鎖定機構）

### 步驟5：精確對準

- [ ] 產品中心與輔助線對齊
- [ ] 產品輪廓與輔助線貼合
- [ ] 多個樣品測試結果一致
- [ ] 偏差<10像素

### 步驟6：切換模式

- [ ] 已退出調機模式
- [ ] 已切換至檢測模式
- [ ] 異常復歸執行成功
- [ ] 系統狀態顯示「就緒」

### 步驟7：測試檢測

- [ ] ROI儲存選項已勾選
- [ ] 已放入10-20個測試樣品
- [ ] 檢測過程順利，無異常
- [ ] 影像已自動儲存

### 步驟8：查看影像

- [ ] 已找到當日日期和料號的資料夾
- [ ] 已查看各站點的ROI影像
- [ ] 影像品質符合標準或已記錄問題

### 步驟9：圓心校正

- [ ] 已開啟圓心校正工具
- [ ] 已載入原始影像（origin資料夾）
- [ ] 已調整參數使輔助線貼合
- [ ] 所有影像偏差<5像素
- [ ] 已儲存調整結果

### 步驟10：再次測試

- [ ] 已執行再次測試
- [ ] 已查看新拍攝的影像
- [ ] 位置偏差在可接受範圍內
- [ ] ROI對齊良好

### 步驟11：完成設定

- [ ] 影像儲存已關閉
- [ ] 所有介面設定已確認正確
- [ ] 已執行最終異常復歸
- [ ] 已開始正式檢測
- [ ] 初期監控（30-60分鐘）無異常
- [ ] 已記錄更換作業

## 附錄

### 更換料號與新增料號的差異總結

| 階段 | 更換料號 | 新增料號 |
|------|----------|----------|
| **料號建立** | 已完成 | 需要建立 |
| **參數設定** | 已完成（載入即用） | 需要完整設定 |
| **硬體調整** | 需要（重點） | 通常不需要 |
| **相機參數** | 已設定，可能需微調 | 需要完整設定 |
| **位置參數** | 需要微調（圓心校正） | 需要完整校正 |
| **檢測參數** | 已設定 | 需要完整設定 |
| **白色像素校正** | 已完成 | 需要執行 |
| **測試驗證** | 需要（驗證對位） | 需要（完整驗證） |

### 關鍵時間節點

| 步驟 | 預計時間 | 備註 |
|------|----------|------|
| 硬體調整（步驟1） | 5-15分鐘 | 取決於產品差異大小 |
| 設定料號（步驟2） | 1分鐘 | 簡單操作 |
| 調機對準（步驟3-5） | 10-20分鐘 | 需要耐心調整 |
| 模式切換（步驟6） | 1分鐘 | 簡單操作 |
| 測試檢測（步驟7-8） | 5-10分鐘 | 取決於樣品數量 |
| 圓心校正（步驟9） | 10-20分鐘 | 需要仔細調整 |
| 再次測試（步驟10） | 5-10分鐘 | 驗證調整效果 |
| 完成設定（步驟11） | 2-5分鐘 | 最終確認 |
| **總計** | **40-85分鐘** | 首次操作偏長，熟練後可縮短 |
