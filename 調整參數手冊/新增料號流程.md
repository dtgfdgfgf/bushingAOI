# 新增料號流程

> **完成時間**：約60-90分鐘 | **前提**：料號尚未建立

---

## 📌 流程總覽

新增料號需要完成 **三大項目**：

1. **初步設定** - 建立料號 + 導入參數 + 測試取像（15-20分鐘）
2. **進階設定** - 校正位置 + 校正檢測參數（40-60分鐘）
3. **瑕疵設定** - 設定要檢測的瑕疵類型（10-15分鐘）

---

## 第一階段：初步設定

### 目標：建立料號並驗證基本拍攝功能

#### 步驟1：新建料號

```
設定 → 料號設定 → 新建料號
```

- 輸入料號代碼（例如：`10214090420T`）
- 輸入料號名稱（選填）

---

#### 步驟2：進入設定模式

- 選擇剛建立的料號
- 點擊介面上的「**設定**」按鈕

---

#### 步驟3：選擇來源料號

```
設定 → 檢測參數設定 → 選擇來源料號
```

**選擇原則：**
- ✅ 第一優先：PTFE **同顏色**（例如：白色 → 白色）
- ✅ 第二優先：**大小接近**（外徑相近±2mm）

---

#### 步驟4：自動導入參數

點擊「**部分自動導入**」按鈕

**會自動導入：**
- ✓ 所有相機參數
- ✓ 所有位置參數
- ✓ 所有時間參數
- ✓ 部分檢測參數

---

#### 步驟5：測試拍攝（驗證delay參數）

**操作順序：**
1. 異常復歸
2. 資料儲存 → 勾選「**test**」
3. 確認介面顯示「**測試中**」
4. 放置樣品
5. 點擊「**開始檢測**」

**目的：** 確認相機參數中的「delay」是否需要調整

---

#### 步驟6：檢查影像位置

```
檔案 → 開啟圖檔 → 查看剛才拍攝的影像
```

**檢查重點：** 產品是否在影像**正中間**？

- ✅ 在正中間 → 繼續步驟7
- ❌ 偏左或偏右 → 回到步驟5，調整delay參數後重新測試

---

#### 步驟7：確認可以正常取像

如果影像位置正確，代表可以開始進行精確的參數調整。

**準備以下影像：**
- 步驟5取得的正常產品影像
- 桌面 `white` 資料夾中的無樣品影像（預先放置）

✅ **第一階段完成**

---

## 第二階段：進階設定

### 目標：精確校正位置和檢測參數

#### 步驟8：位置校正（圓心校正工具）

```
設定 → 檢測參數設定 → 位置參數 → 開啟位置校正工具
```

**詳細操作請參考：** `11.位置參數(圓心校正工具).pdf`

⚠️ **重要提醒：**
- 需要重複做 **12次**（4個站點 × 3種圓）
- **每做完一次，必須點擊「儲存參數」**
- 不要做完一個馬上切到下一個，**確保已儲存**

---

#### 步驟9：檢測參數校正

```
設定 → 檢測參數設定 → 檢測參數 → 校正類型
```

**需要校正5種參數：**

| 編號 | 校正項目 | 參考文件 |
|------|----------|----------|
| 1 | 檢測精度校正 | `20.檢測精度校正(使用原始圖像).md` |
| 2 | 白色像素校正 | `21.白色像素校正(使用原始圖像).md` |
| 3 | 對比檢測校正 | `22.對比檢測校正(使用ROI圖像).md` |
| 4 | 物體偏移校正 | `23.物體偏移校正(使用ROI圖像).md` |
| 5 | 開口閾值校正 | `24.開口閾值校正(使用ROI圖像).md` |

⚠️ **完成後記得點擊右下角「儲存」**

---

#### 步驟10：驗證ROI切圖

**操作順序：**
1. 資料儲存 → **取消勾選** `test`
2. 資料儲存 → **勾選** `ROI`
3. 異常復歸
4. 開始檢測（放入少量樣品）

**目的：** 確認是否正常推料和切圖

---

#### 步驟11：檢查ROI資料夾

**檢查路徑：**
```
bin\x64\Release\report\{料號}\
├── origin\      （原始影像）
├── ROI_1\       （站點1切圖）
├── ROI_2\       （站點2切圖）
├── ROI_3\       （站點3切圖）
└── ROI_4\       （站點4切圖）
```

**檢查方式：**
- 開啟 `ROI_1` ~ `ROI_4` 資料夾
- 確認是否有正常切圖
- 可同時開啟 `origin` 資料夾對比，快速切換視窗查看ROI切取位置

---

#### 步驟12：確認完成

若 ROI 切圖正常，代表進階設定完成 ✅

---

## 第三階段：檢測瑕疵設定

### 目標：設定要檢測的瑕疵類型和閾值

#### 步驟1：開啟檢測瑕疵設定

```
設定 → 檢測瑕疵設定
```

**此處設定：**
- 要檢測哪些瑕疵
- 各瑕疵的閾值
- AOI相關參數

---

#### 步驟2：設定AI檢測參數

**說明：**
- 大部分AOI參數已在「自動導入」時完成
- 使用者只需設定 **AI檢測** 的部分

**可參考其他料號的設定值**

---

#### 步驟3：設定瑕疵種類

⚠️ **重要注意事項：**

| 項目 | 要求 |
|------|------|
| **輸入語言** | 只能輸入**英文** |
| **大小寫** | 不分大小寫 |
| **拼寫** | 務必每個字都打對 |

**不確定瑕疵的中英對照？**
→ 參考其他已存在料號的設定

---

#### 步驟4：儲存設定

確認所有瑕疵設定完成後，點擊「**儲存**」

✅ **新增料號流程完成**

---

## ⚠️ 常見錯誤提醒

| 問題 | 解決方式 |
|------|----------|
| 影像偏左或偏右 | 調整相機參數中的「delay」值 |
| 位置校正沒有儲存 | 每做完一次（一個站點的一種圓）就要點儲存，共12次 |
| ROI切圖位置不對 | 回到步驟8重新進行位置校正 |
| 瑕疵種類設定錯誤 | 檢查拼寫、確認使用英文、參考其他料號 |
| 檢測參數校正完沒儲存 | 記得點右下角「儲存」按鈕 |
