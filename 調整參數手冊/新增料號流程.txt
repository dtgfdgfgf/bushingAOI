一共四項：料號設定+相機參數設定(與檢測參數一起設定)+檢測瑕疵設定+檢測參數設定


初步設定
1. 設定 -> 料號設定 -> 新建料號
2. 選擇該料號，在介面上點擊"設定"
3. 設定 -> 檢測參數設定 -> 選擇PTFE同色的料號為來源料號，若有多種PTFE同色的料號，選擇大小接近的
4. 點擊"部分自動導入"，可自動完成"所有相機參數"、"所有位置參數"、"所有時間參數"、"部分檢測參數"
5. 異常復歸 -> 資料儲存 > 勾選test -> 介面顯示"測試中" -> 放置好樣品後點擊開始檢測 （此步驟是初步取像，確認自動導入的相機參數 "delay" 是否需要調整）
6. 檔案 -> 開啟圖檔，到資料夾中確認影像是否在正中間，若否，重複第五步驟
7. 如果影像在正中間說明可以正常取像，這時候就可以開始進行精確的參數調整，會使用到5. 取到的正常圖像，以及位於桌面white資料夾中，預先放置好的無樣品的影像

進階設定
8. 設定 -> 檢測參數設定 -> 位置參數 -> 開啟位置校正工具 -> 剩餘部份請參考 "11.位置參數(圓心校正工具).pdf"
8.1 過程中需要重複做12次 （四個站點、三種圓），每次做完一種都要點擊儲存參數，所以要點12次，請務必注意不要做完一個馬上切到下一個
9. 檢測參數 -> 校正類型，這裡有五種參數要校正，請參考20~24 全部做完後，記得點擊右下角儲存
10. 資料儲存 > 區消勾選test -> 勾選ROI -> 異常復歸 -> 開始檢測 （少量樣品）-> 看是否正常推料
11. 到資料夾檢查ROI_1~4資料夾，看是否有正常切圖，可同時開啟origin資料夾中的原圖，快速切窗對比ROI切取位置
12. 若一切正常，代表設定完成

檢測瑕疵設定：
1. 位於設定 -> 檢測瑕疵設定，此處是要新增欲檢測之瑕疵以及閥值，以及一些AOI相關參數
2. 大部分AOI相關參數，在自動導入階段都已經導入了，使用者只需要設定AI檢測的部分
3. 設定過程中，可參考其他料號的設定，需要注意的是"瑕疵種類"，務必每個字都打對，不分大小寫（只能輸入英文！）
4. 如果不孰悉瑕疵的中英對照，可參考已存在的料號
