# 物體偏移校正工具使用說明

## 🎯 功能用途

物體偏移校正工具（ObjectBiasCalibration）用於測量並設定產品位置偏移的容許範圍。

系統會測量產品在輸送帶上的位置變化，自動計算合理的偏移容許值（objBias），避免因位置微小變化而誤判為不良品。

---

## 📝 使用時機

**需要使用此工具的情況**：
- 產品在取像位置可能會抖動
- 經常出現「位置偏移」警告
- 良品被誤判為不良（因為位置稍微偏移）

**不需要使用的情況**：
- 產品位置非常穩定
- 使用機械定位裝置
- 複製的參考料號 objBias 值已經適用

---

## 📝 操作步驟

### Step 1：開啟工具

1. 在參數設定視窗中，點擊「**檢測參數**」標籤
2. 點擊「**物體偏移校正**」按鈕（如果有提供）

---

### Step 2：連續拍攝測試影像

1. **準備同一個產品樣品**
   - 使用良品樣品
   - 不要特意移動產品
   - 讓產品自然地通過檢測站

2. **連續拍攝 10-20 張影像**
   - 可以手動觸發拍照
   - 或讓產品連續通過自動拍照
   - 每次都是同一個產品樣品

3. **載入影像到工具**
   - 選擇所有拍攝的影像
   - 工具會自動分析

---

### Step 3：系統自動分析

系統會：
1. **偵測每張影像中產品的位置**
2. **計算位置變化範圍**
   ```
   例如：
   X軸位置：505, 510, 498, 512, 507, ...
   Y軸位置：385, 382, 390, 384, 387, ...
   ```
3. **計算偏移範圍**
   ```
   X軸偏移：498 ~ 512 像素 → 範圍 = 14 像素
   Y軸偏移：382 ~ 390 像素 → 範圍 = 8 像素
   ```

---

### Step 4：查看建議值

系統會顯示：

```
分析結果：
┌─────────────────────────┐
│ X軸偏移範圍：14 像素     │
│ Y軸偏移範圍：8 像素      │
├─────────────────────────┤
│ 建議 objBias 值：        │
│ objBias1 = 20           │
│ objBias2 = 15           │
│                         │
│ (範圍 + 安全餘裕)        │
└─────────────────────────┘

```

---

### Step 5：套用並儲存

1. **檢查建議值是否合理**
   - 通常 objBias 在 10-50 之間
   - 過小（< 5）：容易誤判
   - 過大（> 50）：可能遺漏真正的位置偏移問題

2. **套用建議值**
   - 點擊「**套用建議值**」按鈕
   - 或手動輸入自訂值

3. **儲存參數**
   - 點擊「**儲存**」按鈕
   - 參數會自動新增到「檢測參數」的「已新增已修改區」

---

## ⚠️ 注意事項

1. **使用同一個樣品**
   - 必須使用同一個產品重複拍攝
   - 不要使用不同的產品（會混淆位置變化和產品差異）

2. **樣品數量**
   - 至少 10 張影像
   - 建議 15-20 張（更準確）
   - 過少（< 5 張）：不準確
   - 過多（> 50 張）：沒有必要

3. **測試環境**
   - 在正常生產條件下測試
   - 輸送帶速度正常
   - 照明正常
   - 避免人為干擾

## 💡 使用技巧

### objBias 值的意義

```
objBias 是「位置容許偏移量」：

假設圓心應該在 (500, 400)
objBias1 = 20
objBias2 = 15

則容許範圍：
X軸：500 ± 20 → 480 ~ 520
Y軸：400 ± 15 → 385 ~ 415

只要產品圓心在這個範圍內，都視為位置正確。
```

### 判斷 objBias 是否合理

| objBias 範圍 | 評估 | 建議 |
|-------------|------|------|
| < 5 | 太嚴格 | 可能誤判良品，建議增加 |
| 5-15 | 理想 | 位置非常穩定的系統 |
| 15-30 | 正常 | 一般工業生產線 |
| 30-50 | 偏大 | 可接受，但建議改善定位 |
| > 50 | 太大 | 檢查機械定位系統是否有問題 |

### 改善位置穩定性的方法

如果 objBias 需要設很大（> 50），建議：

1. **改善機械定位**
   ```
   - 加裝定位導軌
   - 調整輸送帶張力
   - 減少輸送帶震動
   ```

2. **調整檢測流程**
   ```
   - 增加穩定延遲時間
   - 調整觸發點位置
   - 使用更精確的觸發方式
   ```

3. **檢查設備**
   ```
   - 輸送帶是否老化
   - 導軌是否鬆動
   - 感測器是否正常
   ```

## 常見問題

### ❓ Q1：objBias 和位置參數有什麼不同？

**答案**：
- **位置參數**（centerX, centerY, radius）：定義產品「應該」在哪裡
- **objBias**：允許產品位置有多少「偏差」

例如：
- 位置參數：圓心應該在 (500, 400)
- objBias = 20：容許圓心在 480-520, 380-420 範圍內

---

### ❓ Q2：每個新料號都需要重新校正 objBias 嗎？

**答案**：
不一定，視情況而定：

**需要重新校正**：
- 產品尺寸差異很大
- 產品形狀不同（定位方式不同）
- 在不同的生產線上生產

**不需要重新校正**：
- 同系列產品，尺寸相近
- 相同的生產線和定位方式
- 可以直接複製參考料號的 objBias 值

---

### ❓ Q3：objBias 設太大會有什麼問題嗎？

**答案**：
會有以下風險：
- 真正的位置偏移問題無法被檢測出來
- 可能遺漏因位置不正確導致的檢測失敗
- 降低檢測準確性

建議：
- objBias 剛好能容納正常位置變化即可
- 不要設得過大「求保險」
- 如果需要很大的 objBias，應該先改善定位系統
